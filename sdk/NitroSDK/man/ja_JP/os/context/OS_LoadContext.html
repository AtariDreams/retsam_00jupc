OS_LoadContext

構文

#include <nitro/os.h>

void OS_LoadContext( OSContext* context );

引数

context コンテキスト

返り値

なし。

説明

指定のコンテキスト構造体からコンテキストを読み込みます。
これはスレッドシステムの、コンテキスト切り替えで使用されます。

コンテキストがロードされると、そのロードされたコンテキストに従ってプロセッサは動作します。

コンテキストの内容については OS_InitContext() を参照してください。

参照

OS_InitContext, OS_SaveContext, OS_DumpContext
