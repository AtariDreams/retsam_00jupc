OS_InitMessageQueue

構文

#include <nitro/os.h>
void OS_InitMessageQueue(
     OSMessageQueue*   mq,
     OSMessage*        msgArray,
     s32               msgCount　);

引数

mq メッセージキューへのポインタ
msgArray メッセージを保持する配列へのポインタ
msgCount 配列が保持できるメッセージ数

返り値

なし。

説明

メッセージキューの構造体mq を初期化します。
引数msgArray は、このキューついてメッセージを保持する配列へのポインタで、引数msgCount は、キューが保持できるメッセージ数 (すなわち、配列のサイズ) です。

参照

OS_JamMessage, OS_SendMessage, OS_ReceiveMessage, OS_ReadMessage
