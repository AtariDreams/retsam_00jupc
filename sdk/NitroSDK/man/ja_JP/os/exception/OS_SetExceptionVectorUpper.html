OS_SetExceptionVectorUpper

構文

#include <nitro/os.h>
void OS_SetExceptionVectorUpper( void );

引数

なし。

返り値

なし。

説明

例外ベクタを上位アドレスへ移動します。

例外ベクタの位置は、32ビットアドレス空間の最下部にある正規アドレス(0x00000000〜0x0000001C)から、アドレス空間の最上部に近いハイベクタ(0xFFFF0000〜0xFFFF001C) に移すことが出来ます。ただしハイベクタがサポートされているかどうかは実装定義となります。(Nitroではサポートされています。)　OS_SetExceptionVectorUpper() では、例外ベクタをハイベクタへと移動します。

内部動作

システム制御コプロセッサ CP15 のレジスタ1を操作します。
