OS_Init

構文

#include <nitro/os.h>
void OS_Init( void );

引数

なし

返り値

なし。

説明

システムを初期化します。
具体的には以下のことを行なっています。

ARM9
アリーナの初期化を行ないます。( OS_InitArena() )
プロセッサ間通信システムの初期化を行ないます。 ( PXI_Init() )
ロックシステムの初期化を行ないます。( OS_InitLock() )
拡張メインメモリアリーナの初期設定を行ないます。 ( OS_InitArenaEx() )
IRQ割り込みテーブルの初期化を行ないます。( OS_InitIrqTable() )
IRQスタックのチェックデータを埋め込みます。( OS_SetIrqStackChecker() )
例外表示システムの初期化を行ないます。( OS_InitException() )
MI 関連の初期化を行ないます。 ( MI_Init() )
Vカウントアラームシステムの初期化を行ないます。 ( OS_InitVAlarm() )
スレッドシステムの初期化を行ないます。 ( OS_InitThread() )
リセットシステムの初期化を行ないます。( OS_InitReset() )
カートリッジライブラリの初期化を行ないます。( CTRDG_Init() )
カードライブラリの初期化を行ないます。( CARD_Init() )
パワー制御システムの初期化を行ないます。 ( PM_Init() )
ARM7
アリーナの初期化を行ないます。( OS_InitArena() )
プロセッサ間通信システムの初期化を行ないます。( PXI_Init() )
ロックシステムの初期化を行ないます。( OS_InitLock() )
IRQ割り込みテーブルの初期化を行ないます。( OS_InitIrqTable() )
例外表示システムの初期化を行ないます。( OS_InitException() )
チックシステムの初期化を行ないます。( OS_InitTick() )
アラームシステムの初期化を行ないます。( OS_InitAlarm() )
スレッドシステムの初期化を行ないます。( OS_InitThread())
リセットシステムの初期化を行ないます。( OS_InitReset() )
カートリッジライブラリの初期化を行ないます。( CTRDG_Init() )

この OS_Init() は通常、ユーザの NitroMain() 関数に制御が移って最初に呼び出される関数となります。特に理由がなければ最初に呼び出してください。

ARM9 側の OS_Init() では、チックシステムの初期化やアラームシステムの初期化は行なっていません。必要ならば、OS_Init() の後に OS_InitTick() 、OS_InitAlarm() を(この順で)呼び出してください。
