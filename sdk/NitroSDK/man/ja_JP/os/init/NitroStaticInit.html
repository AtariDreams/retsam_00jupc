NitroStaticInit

構文

#include <nitro/sinit.h>
static void NitroStaticInit( void );

引数

なし

返り値

なし。

説明

本関数は C 言語でスタティックイニシャライザを利用するために用意されたものです。

1ファイルにつき 1つの NitroStaticInit 関数を作成することができます。常駐モジュールの場合はアプリケーション起動時に、オーバーレイモジュールの場合はオーバーレイのリンク時に呼び出されます。呼び出されるタイミングの詳細は NitroStartUp をご参照ください。
