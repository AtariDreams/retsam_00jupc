OS_ShowAttentionChina

構文

SDK_WEAK_SYMBOL void OS_ShowAttentionChina( const char** isbn );

引数

isbn 健康上の注意画面に表示する ISBN 番号等の文字列を4つ指定します。 文字列のフォーマットはOS_InitChina関数と同じです。

返り値

なし。

説明

OS_InitChina関数で指定された文字列を表示する起動画面処理を実行します。
表示画面実行中に内部で使用したハードウェアリソースは、関数から戻った時点で全て元通りに解放されます。

この関数はOS_InitChina関数の内部から呼び出されます。 弱いシンボルを持つ関数として コンパイルされており、アプリケーション内で新規にOS_ShowAttentionChina関数を定義した場合、新規関数が優先してリンクされます。
