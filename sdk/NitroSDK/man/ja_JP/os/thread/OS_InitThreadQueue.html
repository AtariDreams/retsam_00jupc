OS_InitThreadQueue

構文

#include <nitro/os.h>
void OS_InitThreadQueue( OSThreadQueue* queue );

引数

queue 初期化するスレッドキューへのポインタ。

返り値

なし。

説明

OS_SleepThread関数とOS_WakeupThread関数を含む、他のスレッド関数で使われるスレッドキューを初期化します。

queue には複数のスレッドを登録することができ、これによりOS_WakeupThread() で複数のスレッドを一度に起動状態にすることができます。

参照

OS_CreateThread, OS_WakeupThread
