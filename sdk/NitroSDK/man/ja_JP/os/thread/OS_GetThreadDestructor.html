OS_GetThreadDestructor

構文

#include <nitro/os.h>
OSThreadDestructor OS_GetThreadDestructor( const OSThread* thread );

引数

thread デストラクタを取得するスレッド構造体へのポインタ

返り値

指定のスレッドに設定されているデストラクタ関数へのポインタ

説明

指定のスレッドに対し、現在設定されているデストラクタ関数を取得します。

デストラクタ関数については、OS_SetThreadDestructor() の説明を参照してください。
