OS_VPrintf

構文

#include <nitro/os.h>

void OS_VPrintf( const char *fmt, va_list vlist );

引数

fmt 出力書式文字列(フォーマット文字列)
vlist 可変引数リスト

返り値

なし。

説明

文字列を表示します。フォーマット文字列(出力書式文字列)に従い可変引数リストを与えることが可能です。引数形式が異なることを除いて、OS_Printf() と等価です。

この関数は weak シンボルでコンパイルされますので、ユーザが独自で関数を定義すれば、それで上書きされます。

これはデバッグ用の関数です。最終ROM版(FINALROM)ライブラリでは出力動作を行ないません。

参照

OS_PutString, OS_Printf
