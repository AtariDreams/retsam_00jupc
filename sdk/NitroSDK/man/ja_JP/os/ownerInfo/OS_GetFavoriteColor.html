OS_GetFavoriteColor

構文

#include <nitro/os.h>

static inline GXRgb OS_GetFavoriteColor(int index);

引数

index 色値を取得する「お気に入りの色」の番号を指定します。
このインデックスは「お気に入りの色」総数である 16 色(4bit)でマスクされます。

返り値

指定した番号に対応するGXRgb型の色値を返します。

説明

OSOwnerInfo構造体のメンバ favoriteColor の番号に対応する
「お気に入りの色」の色値を取得します。

参照

OSOwnerInfo OS_FAVORITE_COLOR_*
OS_GetFavoriteColorTable
