OS_GetProtectionRegion*

構文

#include <nitro/os.h>
OS_GetProtectionRegionParam( regionNo );
OS_GetProtectionRegionAddress( regionNo );
OS_GetProtectionRegionSize( regionNo );
(これらの関数はマクロになっています)

引数

なし。

regionNo プロテクションリージョン番号です。0〜7の値で指定してください。

返り値

すべてu32型の値を返します。指定プロテクションリージョンの情報です。

説明

プロテクションリージョンのベースアドレスとサイズに関する設定を取得します。

プロテクションリージョンの設定はシステムの挙動に大きく関わってきます。プロテクションリージョンの仕組みと各領域の使用用途などを理解した上で設定を行なってください。

OS_GetProtectionRegionParam() はregionNoで指定したプロテクションリージョンのベースアドレスと領域サイズ情報のパラメータをCP15から得たままの値で返します。

OS_GetProtectionRegionAddress() はregionNo で指定したプロテクションリージョンのベースアドレスを返します。

OS_GetProtectionRegionSize() はregionNo で指定したプロテクションリージョンの領域サイズ情報を返します。領域サイズ情報は、HW_C6_PR_xxx と定義された値で、xxx の部分がサイズを表します。サイズとして取りえるのは以下の通りです。

4KB 8KB 16KB 32KB 64KB 128KB 256KB 512KB
1MB 2MB 4MB 8MB 16MB 32MB 64MB 128MB 256MB 512MB
1GB 2GB 4GB

これらはすべて関数マクロとなっています。regionNoは0〜7の値で指定してください。この引数をそのまま文字列として使用しますので0〜7の値を入れた変数の指定などは出来ません。0〜7の値そのもので指定してください。

内部動作

この関数自体はマクロですが、最終的にシステム制御コプロセッサ CP15 のレジスタ6を操作します。
