OS_Get*ArenaLo

構文

#include <nitro/os.h>
void* OS_GetArenaLo( OSArenaId id );

void* OS_GetMainArenaLo( void );
void* OS_GetSubPrivArenaLo( void );
void* OS_GetMainExArenaLo( void );
void* OS_GetITCMArenaLo( void );
void* OS_GetDTCMArenaLo( void );
void* OS_GetSharedArenaLo( void );
void* OS_GetWramMainArenaLo( void );
void* OS_GetWramSubArenaLo( void );
void* OS_GetWramSubPrivArenaLo( void );

引数

id アリーナID

返り値

指定のアリーナの下位境界アドレス。

説明

指定のアリーナの下位境界アドレスを取得します。
id はアリーナIDでアリーナを指定します。アリーナIDについては OS_InitArena() を参照してください。

OS_GetMainArenaLo は OS_GetArenaLo( OS_ARENA_MAIN ) の inline です。
OS_GetSubPrivArenaLo は OS_GetArenaLo( OS_ARENA_MAIN_SUBPRIV ) の inline です。
OS_GetMainExArenaLo は OS_GetArenaLo( OS_ARENA_MAIN ) の inline です。
OS_GetITCMArenaLo は OS_GetArenaLo( OS_ARENA_ITCM ) の inline です。
OS_GetDTCMArenaLo は OS_GetArenaLo( OS_ARENA_DTCM ) の inline です。
OS_GetSharedArenaLo は OS_GetArenaLo( OS_ARENA_SHARED ) の inline です。
OS_GetWramMainArenaLo は OS_GetArenaLo( OS_ARENA_WRAM_MAIN ) の inline です。
OS_GetWramSubArenaLo は OS_GetArenaLo( OS_ARENA_WRAM_SUB ) の inline です。
OS_GetWramSubPrivArenaLo は OS_GetArenaLo( OS_ARENA_WRAM_SUBPRIV ) の inline です。

参照

OS_InitArena, OS_SetArenaLo, OS_GetArenaHi
