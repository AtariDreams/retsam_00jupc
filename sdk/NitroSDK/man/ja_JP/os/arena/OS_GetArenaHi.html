OS_Get*ArenaHi

構文

#include <nitro/os.h>
void* OS_GetArenaHi( OSArenaId id );

void* OS_GetMainArenaHi( void );
void* OS_GetSubPrivArenaHi( void );
void* OS_GetMainExArenaHi( void );
void* OS_GetITCMArenaHi( void );
void* OS_GetDTCMArenaHi( void );
void* OS_GetSharedArenaHi( void );
void* OS_GetWramMainArenaHi( void );
void* OS_GetWramSubArenaHi( void );
void* OS_GetWramSubPrivArenaHi( void );

引数

id アリーナID

返り値

指定のアリーナの上位境界アドレス。

説明

指定のアリーナの上位境界アドレスを取得します。
id はアリーナIDでアリーナを指定します。アリーナIDについては OS_InitArena() を参照してください。

OS_GetMainArenaHi は OS_GetArenaHi( OS_ARENA_MAIN ) の inline です。
OS_GetSubPrivArenaHi は OS_GetArenaHi( OS_ARENA_MAIN_SUBPRIV ) の inline です。
OS_GetMainExArenaHi は OS_GetArenaHi( OS_ARENA_MAINEX ) の inline です。
OS_GetITCMArenaHi は OS_GetArenaHi( OS_ARENA_ITCM ) の inline です。
OS_GetDTCMArenaHi は OS_GetArenaHi( OS_ARENA_DTCM ) の inline です。
OS_GetSharedArenaHi は OS_GetArenaHi( OS_ARENA_SHARED ) の inline です。
OS_GetWramMainArenaHi は OS_GetArenaHi( OS_ARENA_WRAM_MAIN ) の inline です。
OS_GetWramSubArenaHi は OS_GetArenaHi( OS_ARENA_WRAM_SUB ) の inline です。
OS_GetWramSubPrivArenaHi は OS_GetArenaHi( OS_ARENA_WRAM_SUBPRIV ) の inline です。

参照

OS_InitArena, OS_SetArenaHi, OS_GetArenaLo
