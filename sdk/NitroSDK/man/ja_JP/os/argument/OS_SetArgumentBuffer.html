OS_SetArgumentBuffer

構文

#include <nitro/os.h>
void OS_SetArgumentBuffer( const char* buffer );

引数

なし。

buffer 設定する引数バッファへのポインタ

返り値

なし。

説明

OS_GetArgc や OS_GetArgv や OS_GetOpt などで使われる引数バッファを設定します。

デフォルトでは、システム内で持っている引数バッファ領域 ( buryarg ツールによって引数データが埋め込まれます) が用いられますが、この関数で変更することが出来ます。

引数データは OS_ConvertToArguments() を使えば、文字列から容易に変換出来ます。

FINALROM ビルドでは何もしません。

参照

OS_GetArgv, OS_GetArgc, OS_GetOpt, buryarg ツール,
OS_GetArgumentBuffer, OS_ConvertToArguments
