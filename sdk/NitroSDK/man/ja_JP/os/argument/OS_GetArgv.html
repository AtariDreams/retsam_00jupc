OS_GetArgv

構文

#include <nitro/os.h>
cosnt char* OS_GetArgv( int n );

引数

n　　　 取得する引数文字列のインデックス番号

返り値

引数文字列へのポインタです。

説明

ROM ファイル中に埋め込まれた引数データから、指定インデックスの引数文字列を取得します。

引数は、buryarg ツールによって ROM ファイル中に引数データとして埋め込まれます。buryarg を実行したときに指定したROM ファイル名が インデックス0 のデータとなります。

n として OS_GetArgc() で取得出来る値以上の値を指定すると、NULL を返します。負数だった場合の動作は不定です。

FINALROMビルドでは、n に依らず、必ずNULL を返します。

(例)


　　　　　　buryarg main.srl test 1 2 3 と指定していた場合、
　　　　　　　　OS_GetArgc() … 4
　　　　　　　　OS_GetArgv( 0 ) … main.srl
　　　　　　　　OS_GetArgv( 1 ) … test
　　　　　　　　OS_GetArgv( 2 ) … 1
　　　　　　　　OS_GetArgv( 3 ) … 2
　　　　　　　　OS_GetArgv( 4 ) … 3
　　　　　　　　OS_GetArgv( 5 ) … NULL
　　　　　　となります。
