OS_SetCurrentHeap

構文

#include <nitro/os.h>
OSHeapHandle OS_SetCurrentHeap( OSArenaId id, OSHeapHandle heap );

引数

id カレントヒープを設定するアリーナのアリーナID
heap カレントヒープに設定するヒープのハンドル

返り値

以前カレントヒープだったヒープのハンドルです。

説明

カレントヒープを設定します。

カレントヒープはアリーナごとに存在します。この関数では指定のアリーナのカレントヒープのみを設定します。
アリーナはアリーナID id で指定します。この値については OS_InitArena() を参照してください。

参照

OS_InitArena, OS_InitAlloc, OS_CreateHeap
