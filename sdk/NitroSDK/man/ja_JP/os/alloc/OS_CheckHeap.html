OS_CheckHeap

構文

#include <nitro/os.h>
long OS_CheckHeap( OSArenaId id, OSHeapHandle heap );

引数

id チェックするヒープのあるアリーナのアリーナID
heap チェックするヒープのハンドル

返り値

ヒープに矛盾があれば、-1を返します。
そうでなければ、フリーリストでの利用可能なバイト数を返します。.

説明

指定のヒープをチェックします。
これはデバッグ用の関数です。

チェックするヒープのあるアリーナを指定する必要があります。アリーナはアリーナID id で指定します。この値については OS_InitArena() を参照してください

参照

OS_InitArena, OS_InitAlloc, OS_DumpHeap
