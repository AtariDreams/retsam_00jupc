OS_InitReset

構文

#include <nitro/os.h>

void OS_InitReset( void );

引数

なし。

返り値

なし。

説明

リセットシステムを初期化します。

OS_InitReset() は、リセットシステムの関数を呼ぶ前に一度だけ呼ばれている必要があります。ただし、アプリケーションがOS_Init() を呼んでいる場合はその中から呼ばれますので、アプリケーションで改めて呼びなおす必要はありません。

OS_GetResetParameter() に限り、OS_InitReset() の前に呼び出しても構いません。

参照

OS_ResetSystem, OS_GetResetParameter
