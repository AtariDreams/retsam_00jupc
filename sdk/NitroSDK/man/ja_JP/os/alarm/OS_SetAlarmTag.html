OS_SetAlarmTag

構文

#include <nitro/os.h>
void OS_SetAlarmTag( OSAlarm* alarm, u32 tag );

引数

alarm タグを設定するアラーム構造体へのポインタ
tag 設定するタグ値

返り値

なし。

説明

アラームにタグを設定します。
タグは OS_CancelAlarms() で指定のタググループのアラームをキャンセルするのに使用されます。

タグ値「0」はシステムで予約されていますので、ユーザが使用することは出来ません。

参照

OS_InitAlarm, OS_CancelAlarms
