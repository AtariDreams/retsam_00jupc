SVC_IsMmemExpanded

構文

#include <nitro/os/common/systemCall.h>
BOOL SVC_IsMmemExpanded( void );

引数

なし。

返り値

メインメモリが8Mバイトへ拡張されているなら TRUE を返します。

説明

メインメモリが8Mバイトへ拡張されているかどうかを調べます。
メインメモリが拡張されているなら TRUE を、そうでなければFALSE を返します。

ARM9 側では、0x023FFFF8 番地と 0x027FFFF8 番地をキャッシュ無効の設定にしておく必要があります。
