FSSeekFileMode

定義

#include <nitro/fs.h>

typedef enum
{
   FS_SEEK_SET,   /* seek from begin */
   FS_SEEK_CUR,   /* seek from current */
   FS_SEEK_END    /* seek from end */
} FSSeekFileMode;

説明

FS_SeekFile関数で指定する基準位置を表現する列挙型です。
以下のように定義されます。
FS_SEEK_SET 基準位置としてファイルの先頭を指定します。
FS_SEEK_CUR 基準位置としてファイルの現在位置を指定します。
FS_SEEK_END 基準位置としてファイルの終端を指定します。

参照

FS_SeekFile FS_SeekFileToBegin FS_SeekFileToEnd
