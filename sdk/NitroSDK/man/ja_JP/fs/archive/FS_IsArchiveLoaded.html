FS_IsArchiveLoaded

構文

#include <nitro/fs.h>

static inline BOOL FS_IsArchiveLoaded(volatile const FSArchive *p_arc);

引数

p_arc FSArchive構造体のアドレス

返り値

アーカイブがシステムにロードされているなら TRUE を、そうでないなら FALSE を返します。

説明

指定したアーカイブが現在ファイルシステムにロード済みか判定します。

FSArchive
FS_LoadArchive FS_UnloadArchive
