FS_UnloadOverlay

構文

#include <nitro/fs.h>

BOOL FS_UnloadOverlay(MIProcessor target, FSOverlayID id);

引数

target アンロードする対象（ARM9プロセッサ/ARM7プロセッサ）
id アンロードするモジュール

返り値

モジュールが正しくアンロードされれば TRUE を返し、そうでない場合は、FALSE を返します。

説明

指定したプロセッサの指定した番号に関連付けられたオーバーレイモジュールをアンロードします。

この関数は、FS_LoadOverlayInfo関数でオーバーレイモジュール情報を取得して
FS_UnloadOverlayImage関数を呼び出すことと等価です。

参照

FSOverlayInfo FSOverlayID FS_OVERLAY_ID
FS_LoadOverlayInfo FS_UnloadOverlayImage FS_LoadOverlay
