FS_SeekDir

構文

#include <nitro/fs.h>

BOOL FS_SeekDir( FSFile *p_dir, const FSDirPos *p_pos );

引数

p_dir FSFile構造体のアドレス
p_pos ディレクトリリストの位置を格納した FSDirPos 構造体

返り値

ディレクトリリストの位置が正しく取得できれば TRUE を、そうでないなら FALSE を返します。

説明

ディレクトリリストを、列挙中の位置も含めて取得します。
位置を示すFSDirPosの値は、以下の方法で取得します。

FS_ReadDir関数 取得したFSDirEntry構造体のis_directoryメンバが1であれば、
dir_idメンバはディレクトリリストの先頭を表しています。
FS_TellDir関数 成功時、ディレクトリリストの現在位置が格納されています。

注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
