FS_WriteFile

構文

#include <nitro/fs.h>

s32 FS_WriteFile(FSFile* p_file, const void *src, s32 len);

引数

p_file FSFile構造体のアドレス
src 書き込みデータが格納されたバッファのアドレス
len 書き込みデータのバイト長

返り値

データが正しく書き込めればそのバイト数を, そうでないなら -1 を返します。

説明

ファイルポインタの現在位置から指定サイズのデータを書き込みます。
終端を越えるようにサイズ指定した場合、通常は終端まで書き込んでそのサイズを返します。
アーカイブの実装によっては、自動的にサイズ拡張を行うことができる場合があります。

注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
