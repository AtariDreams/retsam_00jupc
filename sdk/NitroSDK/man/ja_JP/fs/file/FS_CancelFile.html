FS_CancelFile

構文

#include <nitro/fs.h>

void FS_CancelFile( FSFile *p_file );

引数

p_file FSFile構造体のアドレス

返り値

なし

説明

指定したファイルが現在非同期処理を実行中であればキャンセルを要求します。
関数は、処理の完了を待たずにすぐ制御を返します。
キャンセルが受理されるかまたは実際に処理が完了するまで待機するには、FS_WaitAsync関数を使用します。

指定したファイルがアーカイブ内で未処理コマンドとして保留されている場合、
アーカイブが現在のコマンドを完了したタイミングでキャンセルが受理されます。
指定したファイルが現在アーカイブ内で処理対象コマンドとなっている場合、
処理をキャンセルするか完遂するかはアーカイブの実装に依存します。
