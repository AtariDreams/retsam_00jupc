FS_ChangeDir

構文

#include <nitro/fs.h>

BOOL FS_ChangeDir( const char *dir );

引数

dir 指定するカレントディレクトリのパス

返り値

カレントディレクトリが正しく設定されれば TRUE を, そうでないなら FALSE を返します。

説明

指定したパス名にカレントディレクトリを設定します。
パス名の形式についてはFS_OpenFile関数の説明を参照下さい。

カレントディレクトリは、実際に存在するもののみ設定可能です。

参照

FS_OpenFile FS_FindDir FS_FILE_NAME_MAX
