FS_IsSucceeded

構文

#include <nitro/fs.h>

static inline BOOL FS_IsSucceeded(volatile const FSFile *p_file);

引数

p_file FSFile構造体のアドレス

返り値

最後に完了した非同期処理が成功なら TRUE を、そうでないなら FALSE を返します。

説明

指定したファイルが最後に完了した非同期処理が正しく実行されたか判定します。
ファイルがすでに閉じられていたり、何らかの不正な処理を行われた場合に FALSE となります。

FSFile FS_ReadFileAsync FS_WaitAsync
