FS_CloseFile

構文

#include <nitro/fs.h>

BOOL FS_CloseFile( FSFile *p_file );

引数

p_file FSFile構造体のアドレス

返り値

ファイルを正しく閉じれば TRUE を, そうでないなら FALSE を返します。

説明

開いているファイルを閉じます。


注意

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
