FS_OpenFile

構文

#include <nitro/fs.h>

BOOL FS_OpenFile( FSFile *p_file, const char *path );

引数

p_file FSFile構造体のアドレス
path ファイルのパス名

返り値

ファイルが正しく開ければ TRUE を, そうでないなら FALSE を返します。

説明

指定したパス名でファイルを開きます。

パス名は以下の形式で指定できます。

・ '/' で始まる絶対パス (カレントディレクトリを無視します)
・ '/' 以外で始まる相対パス (カレントディレクトリを基準にします)

また、パス名に使用できる文字は、ASCIIコードのみです。
さらに、次の文字は禁止されています。 【 \ / : ; * ? " < > | 】

ファイル名およびディレクトリ名の文字数は FS_FILE_NAME_MAX を越えてはいけません。

ディレクトリ名のうち、以下のものは予約されています。

. 現在のディレクトリを示します。(無視されます。)
.. 1階層上のディレクトリを示します。

注意

アプリケーションは、シンボル NITRO_MAKEROM を有効にして ROMファイルシステムを含んでいる必要があります。

この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
