FS_End

構文

#include <nitro/fs.h>

void FS_End( void );

引数

なし

返り値

なし

説明

FSライブラリを終了します。
以下の処理を行います。

  • ロードされている全てのアーカイブに対してFS_UnloadArchive関数およびFS_ReleaseArchiveName関数を呼び出して解放
  • FSライブラリをFS_Init関数が呼ばれる前の状態に戻す

注意

この関数の内部処理は、FSライブラリ終了処理以外の詳細を考慮しません。
FS_LoadTable関数やFS_LoadArchiveTables関数で割り当てたメモリの解放などを含め,
固有の終了処理を必要とするアーカイブが存在する場合はこの関数の呼び出しの前にそれらを独自に解放しておく必要があります。
