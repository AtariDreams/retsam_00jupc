SND_GetWaveDataAddress

構文

#include <nitro/snd.h>
const SNDWaveData* SND_GetWaveDataAddress(
        const SNDWaveArc* waveArc,
        int index );

引数

waveArc 波形アーカイブのポインタです。
index 波形データのインデックス番号です。

返り値

SNDWaveData型の波形データのアドレスを取得します。

説明

波形アーカイブ中の波形データのアドレスを取得します。
