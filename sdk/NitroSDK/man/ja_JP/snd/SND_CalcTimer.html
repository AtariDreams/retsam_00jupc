SND_CalcTimer

構文

#include <nitro/snd.h>
u16 SND_CalcTimer( int orgTimer, int pitch );

引数

orgTimer オリジナルのタイマー値です。
pitch ピッチ変化の値です。

返り値

ピッチ変換後のタイマー値を返します。

説明

ピッチ変換後のタイマー値を計算します。

ピッチ変換の値pitchは、正の時にタイマー周期が早くなり、負の時にタイマー周期が遅くなります。 64の時ちょうど半音分に相当する変化、すなわちタイマー周波数が 2 ** ( 1 / 12 ) 倍になります。

計算式は、次のようになります。 

orgTimer ÷ ( 2 ** ( pitch / ( 64 × 12 ) ) )
