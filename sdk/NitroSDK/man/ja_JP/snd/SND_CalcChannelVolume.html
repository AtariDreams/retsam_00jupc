SND_CalcChannelVolume

構文

#include <nitro/snd.h>
u16 SND_CalcChannelVolume( int dB );

引数

dB デシベル値です。単位は0.1dBで、値の範囲は、-723〜0 (-72.3dB 〜 0.0dB) です。

返り値

チャンネルボリューム値を16ビットで返します。下位8ビットにはボリューム値が、上位8ビットにはデータシフト値が格納されています。

説明

チャンネルボリューム値を計算します。

SND_SetChannelVolume関数などで 直接扱えるボリューム値を取得することができます。 返り値の下位８ビットには、0〜127の範囲のボリューム値が入ります。 上位８ビットには、SNDChannelDataShift型の値が入ります。
