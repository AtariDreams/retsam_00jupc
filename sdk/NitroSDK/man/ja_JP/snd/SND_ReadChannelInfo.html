SND_ReadChannelInfo

構文

#include <nitro/snd.h>
BOOL SND_ReadChannelInfo(
        const SNDDriverInfo* driverInfo,
        int chNo,
        SNDChannelInfo* chInfo );

引数

driverInfo ドライバ情報構造体のポインタです。
chNo チャンネル番号です。値の範囲は、0〜15です。
chInfo 取得した情報を格納するチャンネル情報構造体のポインタです。

返り値

情報の取得に成功したかどうかを返します。

説明

ドライバ情報構造体から、チャンネル情報を取得します。

ドライバ情報構造体driverInfoは、 SND_ReadDriverInfo関数で 取得することができます。

情報の取得に成功すると、SNDChannelInfo型のチャンネル情報構造体chInfoへ、チャンネルの情報が書き込まれます、

参照

SND_ReadDriverInfo, SNDChannelInfo
