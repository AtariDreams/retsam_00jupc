SND_AssignWaveArc

構文

#include <nitro/snd.h>
void SND_AssignWaveArc(
        SNDBankData* bank,
        int index,
        SNDWaveArc* waveArc );

引数

bank バンクデータのポインタです。32バイトアライメントされている必要があります。
index 何番の波形アーカイブとして登録するかを示したインデックス番号です。値の範囲は0〜3です。
waveArc 波形アーカイブのポインタです。32バイトアライメントされている必要があります。

返り値

なし。

説明

バンクbankに、index番の波形アーカイブとして、 waveArcを割り付けます。

バンクbankのindex番に、 別の波形アーカイブがすでに割り付けられていた場合、 一旦その割付を解除した後、指定した波形アーカイブwaveArcを割り付けます。

bank、waveArc共に、 32バイトアライメントされている必要があります。

参照

なし。
