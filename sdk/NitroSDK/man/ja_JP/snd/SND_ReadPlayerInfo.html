SND_ReadPlayerInfo

構文

#include <nitro/snd.h>
BOOL SND_ReadPlayerInfo(
        const SNDDriverInfo* driverInfo,
        int playerNo,
        SNDPlayerInfo* playerInfo );

引数

driverInfo ドライバ情報構造体のポインタです。
playerNo プレイヤー番号です。値の範囲は、0〜15です。
playerInfo 取得した情報を格納するプレイヤー情報構造体のポインタです。

返り値

情報の取得に成功したかどうかを返します。

説明

ドライバ情報構造体から、プレイヤー情報を取得します。

ドライバ情報構造体driverInfoは、 SND_ReadDriverInfo関数で 取得することができます。

情報の取得に成功すると、SNDPlayerInfo型のプレイヤー情報構造体playerInfoへ、プレイヤーの情報が書き込まれます、

参照

SND_ReadDriverInfo, SNDPlayerInfo
