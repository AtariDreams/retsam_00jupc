グラフィックス(GX/G2/G3/G3X) API 機能一覧

2D/3Dグラフィックス全般(GX)

2Dグラフィックスエンジン共通

パワーコントロール

GX_SetPower 各種パワーコントロールを一括設定します。
GX_GetPower 各種パワーコントロールの状態を取得します。
GX_Power2D メイン・サブの２ＤエンジンのON/OFFを設定します。
GX_Power3D ジオメトリエンジンとレンダリングエンジンのON/OFFを設定します。
GX_Power2DMain メイン２ＤエンジンのON/OFFを設定します。
GX_Power2DSub サブ２ＤエンジンのON/OFFを設定します。
GX_PowerGeometry ジオメトリエンジンのON/OFFを設定します。
GX_PowerRender レンダリングエンジンのON/OFFを設定します。
GX_SetDispSelect LCD出力先を設定します。
GX_GetDispSelect LCD出力先の設定を取得します。

初期化

GX_Init グラフィックスシステムを初期化します。
GX_InitEx グラフィックスシステムを初期化します。またGXのデフォルトDMA番号の変更をおこないます。
GX_SetDefaultDMA GXの使用するDMA番号の設定を変更します。
GX_GetDefaultDMA GXの使用するDMA番号の設定を取得します。

VRAMバンクのリソースへの割り当て

GX_SetBankForBG VRAMバンクを指定してメイン2DエンジンのBGに割り当てます。
GX_SetBankForBGEx 連続的なアドレスに割り当てることができない組み合わせのVRAMバンクを指定してメイン2DエンジンのBGに割り当てます。
GX_SetBankForOBJ VRAMバンクを指定してメイン2DエンジンのOBJに割り当てます。
GX_SetBankForBGExtPltt VRAMバンクを指定してメイン2DエンジンのBG拡張パレットに割り当てます。
GX_SetBankForOBJExtPltt VRAMバンクを指定してメイン2DエンジンのOBJ拡張パレットに割り当てます。
GX_SetBankForTex VRAMバンクを指定してテクスチャイメージスロットに割り当てます。
GX_SetBankForTexPltt VRAMバンクを指定してテクスチャパレットスロットに割り当てます。
GX_SetBankForClearImage VRAMバンクを指定してクリアイメージに割り当てます。
GX_SetBankForSubBG VRAMバンクを指定してサブ2DエンジンのBGに割り当てます。
GX_SetBankForSubOBJ VRAMバンクを指定してサブ2DエンジンのOBJに割り当てます。
GX_SetBankForSubBGExtPltt VRAMバンクを指定してサブ2DエンジンのBG拡張パレットに割り当てます。
GX_SetBankForSubOBJExtPltt VRAMバンクを指定してサブ2DエンジンのOBJ拡張パレットに割り当てます。
GX_SetBankForARM7 VRAMバンクを指定してＡＲＭ７メモリ空間に割り当てます。
GX_SetBankForLCDC VRAMバンクを指定してLCDCメモリ空間に割り当てます。
GX_TrySetBankForBG メイン2DエンジンのBGへVRAMバンクの割り当てを試みます。
GX_TrySetBankForBGEx メイン2DエンジンのBGへ、連続的なアドレスに割り当てることができない組み合わせのVRAMバンクの割り当てを試みます。
GX_TrySetBankForOBJ メイン2DエンジンのOBJへVRAMバンクの割り当てを試みます。
GX_TrySetBankForBGExtPltt メイン2DエンジンのBG拡張パレットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForOBJExtPltt メイン2DエンジンのOBJ拡張パレットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForTex テクスチャイメージスロットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForTexPltt テクスチャパレットスロットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForClearImage クリアイメージへVRAMバンクの割り当てを試みます。
GX_TrySetBankForSubBG サブ2DエンジンのBGへVRAMバンクの割り当てを試みます。
GX_TrySetBankForSubOBJ サブ2DエンジンのOBJへVRAMバンクの割り当てを試みます。
GX_TrySetBankForSubBGExtPltt サブ2DエンジンのBG拡張パレットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForSubOBJExtPltt サブ2DエンジンのOBJ拡張パレットへVRAMバンクの割り当てを試みます。
GX_TrySetBankForARM7 ＡＲＭ７メモリ空間へVRAMバンクの割り当てを試みます。
GX_TrySetBankForLCDC LCDCメモリ空間へVRAMバンクの割り当てを試みます。
