GX_SetDispSelect

構文

#include <nitro/gx/gx.h>

void GX_SetDispSelect(GXDispSelect sel);

引数

sel LCD出力先の割り当て

返り値

なし。

説明

LCD出力先を設定します。

GX_DISP_SELECT_MAIN_SUB 上のLCDにメインエンジンの出力（２Ｄ／３Ｄ）を、下のLCDにサブエンジンの出力（２Ｄ）を表示します。
GX_DISP_SELECT_SUB_MAIN 上のLCDにサブエンジンの出力（２Ｄ）を、下のLCDにメインエンジンの出力（２Ｄ／３Ｄ）を表示します。


以下にGXDispSelect型の型定義を示しておきます。

typedef enum
{
    GX_DISP_SELECT_SUB_MAIN = 0,
    GX_DISP_SELECT_MAIN_SUB = 1
}
GXDispSelect;
