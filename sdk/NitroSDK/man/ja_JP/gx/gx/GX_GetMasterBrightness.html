GX*_GetMasterBrightness

構文

#include <nitro/gx/gx.h>

int GX_GetMasterBrightness( void );
int GXS_GetMasterBrightness( void );

引数

なし。

返り値

輝度を調整するための値(-16以上16以下)

説明

画像出力において、輝度調整パラメータを取得します。-16から16までの整数が取得でき、値が正であれば輝度は上がり、値が負であれば輝度は下がっています。
GX_GetMasterBrightness関数ではメイングラフィックエンジンの設定を取得し、GXS_GetMasterBrightness関数ではサブグラフィックエンジンの設定を取得します。

参照

G2_SetBlendBrightness, GX_SetMasterBrightness, GXS_SetMasterBrightness
