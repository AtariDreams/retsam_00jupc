GXDLInfo

定義

#include <nitro/gx/g3.h>
typedef struct
{
    u8 *curr_cmd;
    u32 *curr_param;
    u32 *bottom;
    u32 length;
    BOOL param0_cmd_flg;
}
GXDLInfo;

要素

curr_cmd コマンド挿入位置へのポインタ
curr_param コマンドパラメータ挿入位置へのポインタ
bottom ディスプレイリスト末尾へのポインタ
length ディスプレイリストのサイズ
param0_cmd_flg パラメータ0のジオメトリコマンド送信時のハードウェアバグ対策パラメータ

説明

G3B_***, G3C_***等の動的ディスプレイリスト作成関数の引数として使われる構造体です。
