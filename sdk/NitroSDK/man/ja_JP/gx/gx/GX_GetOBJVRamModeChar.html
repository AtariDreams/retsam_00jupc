GX*_GetOBJVRamModeChar

構文

#include <nitro/gx/gx.h>

GXOBJVRamModeChar GX_GetOBJVRamModeChar( void );
GXOBJVRamModeChar GXS_GetOBJVRamModeChar( void );

引数

なし。

返り値

マッピングモードを取得します。

説明

キャラクタOBJのOBJ-VRAM容量とマッピングモードを取得します。GX_GetOBJVRamModeChar関数ではメイングラフィックスエンジンの、GXS_GetOBJVRamModeCharではサブグラフィックスエンジンの設定が取得されます。



GXOBJVRamModeChar型の型定義はGX_SetOBJVRamModeCharを参照してください。

参照

GX_GetOBJVRamModeBmp, GX_SetOBJVRamModeChar, GXS_SetOBJVRamModeChar
