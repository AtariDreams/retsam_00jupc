GX_SetVCount

構文

#include <nitro/gx/gx.h>

void GX_SetVCount(s32 count);

引数

count Vカウンタに設定するVカウント(202 <= count <= 211)

返り値

なし

説明

Vカウンタレジスタに値を書き込みます。書き込んだ値は、Vカウンタ更新時に反映されます。現在のVカウント値が202から211であることを確認した上で202から211の値を書き込んでください。
ただし、この関数の呼び出し中に 212 ラインになってしまった場合はVカウントを設定することができませんので、 確実を期すためには割り込み禁止にした上で 210 ラインまでに呼び出す必要があります。

注意：LCD に悪影響を及ぼす危険性があるため、以下の使用方法を禁止します。

  • Vカウントを進める。
  • 1フレームに2回以上呼ぶ。

また、MP 通信中にこの関数を使用してVカウントを設定すると親子のVブランク同期が乱れ、 MP通信のフレーム同期通信モードでの通信が安定して行えなくなりますので、MP 通信中はこの関数は使用しないでください。
