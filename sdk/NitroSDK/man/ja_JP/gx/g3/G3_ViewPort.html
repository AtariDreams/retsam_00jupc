G3*_ViewPort

構文

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_ViewPort(int x1, int y1, int x2, int y2);

// 以下ディスプレイリスト（コマンド列）作成用API
void G3B_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3C_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3BS_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);
void G3CS_ViewPort(GXDLInfo* info, int x1, int y1, int x2, int y2);

引数

info コマンドリスト情報
x1 左下のＸ座標
y1 左下のＹ座標
x2 右上のＸ座標
y2 右上のＹ座標

返り値

なし。

説明

ビューポートを設定します。ビューポートの設定座標値は、X座標は0〜255、Y座標は0〜191です。
