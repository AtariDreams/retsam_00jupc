G3*_LightColor

構文

#include <nitro/gx/g3imm.h>
#include <nitro/gx/g3b.h>
#include <nitro/gx/g3c.h>

void G3_LightColor(GXLightId lightID, GXRgb rgb);

// 以下ディスプレイリスト（コマンド列）作成用API
void G3B_LightColor(GXDLInfo* info, GXLightId lightID, GXRgb rgb);
void G3C_LightColor(GXDLInfo* info, GXLightId lightID, GXRgb rgb);
void G3BS_LightColor(GXDLInfo* info, GXLightId lightID, GXRgb rgb);
void G3CS_LightColor(GXDLInfo* info, GXLightId lightID, GXRgb rgb);

引数

info コマンドリスト情報
lightID ライトID
rgb ライトカラー

返り値

なし。

説明

ライトカラーを設定します。lightIDで4つのうちから1つライトを選び、rgbでカラーを設定します。

GX_LIGHTID_0 ライト0を指定
GX_LIGHTID_1 ライト1を指定
GX_LIGHTID_2 ライト2を指定
GX_LIGHTID_3 ライト3を指定


以下にGXLightId型の型定義を示しておきます。

typedef enum
{
    GX_LIGHTID_0 = 0,
    GX_LIGHTID_1 = 1,
    GX_LIGHTID_2 = 2,
    GX_LIGHTID_3 = 3
}
GXLightId;
