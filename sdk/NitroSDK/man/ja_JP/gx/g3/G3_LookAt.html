G3_LookAt

構文

#include <nitro/gx/g3_util.h>

void G3_LookAt(
    const VecFx32 * camPos,
    const VecFx32 * camUp,
    const VecFx32 * target,
    MtxFx43 * mtx
);

引数

camPos カメラの位置ベクトルへのポインタ
camUp カメラの上方向のベクトルへのポインタ
target カメラの焦点へのポインタ
mtx 4x3行列へのポインタ

返り値

なし。

説明

行列モードをPosition/Vectorモードに設定し、カレント行列にカメラ行列をセットします。mtxがNULLでなければ*mtxにもカメラ行列をセットします。*camUpは正規化されている必要はありません。
