G3_MakeTexPlttBaseParam

構文

#include <nitro/gx/g3.h>

static inline 
GXTexPlttBaseParam G3_MakeTexPlttBaseParam( u32 plttBaseAddr, GXTexFmt texfmt )

引数

plttBaseAddr パレットのベースアドレス
texfmt テクスチャのフォーマット

返り値

REG_G3TEXPLTT_BASEへ設定するアドレス値

説明

テクスチャパレットのベースアドレスをテクスチャパレットベースレジスタ(REG_G3_TEXPLTT_BASE)へ設定する値に変換するための関数です。G3*_TexPlttBase関数を使用する場合には必要ありませんが、texfmtが定数として与えられた場合には最適化により高速に実行されます。
