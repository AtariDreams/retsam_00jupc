G3X_SetFifoIntrCond

構文

#include <nitro/gx/g3x.h>

void G3X_SetFifoIntrCond(GXFifoIntrCond cond);

引数

コマンドFIFO割り込み要求発生条件

返り値

なし

説明

コマンドFIFO割り込み要求発生条件を設定します。割り込み要求不可・半分未満で割り込み要求・空で割り込み要求の3つのオプションから選択します。

GX_FIFOINTR_COND_DISABLE コマンドＦＩＦＯ割り込み要求不可
GX_FIFOINTR_COND_UNDERHALF コマンドＦＩＦＯが半分以下のときに割り込み要求
GX_FIFOINTR_COND_EMPTY コマンドFIFOが空のときに割り込み要求


以下にGXFifoIntrCond型の型定義を記しておきます。

typedef enum
{
    GX_FIFOINTR_COND_DISABLE = 0,
    GX_FIFOINTR_COND_UNDERHALF = 1,
    GX_FIFOINTR_COND_EMPTY = 2
}
GXFifoIntrCond;
