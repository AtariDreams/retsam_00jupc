G3X_GetCommandFifoStatus

構文

#include <nitro/gx/g3x.h>

GXFifoStat G3X_GetCommandFifoStatus(void);

引数

なし

返り値

コマンドFIFOの大まかな状態

説明

コマンドFIFOの状態を、空・半分以下・半分以上・一杯の4種類の状態として返します。

GX_FIFOSTAT_EMPTY コマンドFIFOは空です。
GX_FIFOSTAT_UNDERHALF コマンドFIFOは半分以下ですが空ではありません。
GX_FIFOSTAT_OVERHALF コマンドFIFOは半分以上ですが一杯ではありません。
GX_FIFOSTAT_FULL コマンドFIFOは一杯です。


以下にGXFifoStat型の型定義を記しておきます。

typedef enum
{
    GX_FIFOSTAT_EMPTY = 6,
    GX_FIFOSTAT_UNDERHALF = 2,
    GX_FIFOSTAT_OVERHALF = 0,
    GX_FIFOSTAT_FULL = 1
}
GXFifoStat;
