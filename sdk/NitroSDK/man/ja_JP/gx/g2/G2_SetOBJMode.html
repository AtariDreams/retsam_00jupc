G2_SetOBJMode

構文

#include <nitro/gx/g2_oam.h>

void G2_SetOBJMode(GXOamAttr * oam, GXOamMode mode, int cParam);

引数

oam OAMデータの格納場所へのポインタ
mode OBJモード
cParam カラーパラメータ

返り値

なし。

説明

*oamに格納されているOAMデータのOBJモードとカラーパラメータを変更します。cParamはOBJモードがビットマップOBJ以外の場合のとき、16パレット中1パレットを指定します。このとき、OBJ拡張パレットがロードされておらず、カラーモードが256色モードのときは設定は無視されます。OBJモードがビットマップOBJの場合は、OBJのα値になります。GXOamMode型についてはG2_SetOBJAttrを参照してください。
