G2*_GetBG3ControlDCBmp

構文

#include <nitro/gx/gx_bgcnt.h>
static inline GXBg23ControlDCBmp G2_GetBG3ControlDCBmp( void );
static inline GXBg23ControlDCBmp G2S_GetBG3ControlDCBmp( void );

引数

なし。

返り値

BG3の設定。

説明

アフィン拡張BGでダイレクトカラービットマップBGの場合のBG3の設定をGXBg23ControlDCBmp構造体として取得します。
各メンバについてはG2_SetBG3ControlDCBmpを参照してください。

この関数はBG3がダイレクトカラービットマップBGに設定されている状態で使用する必要があります。

G2_GetBG3ControlDCBmp関数ではメイン2Dエンジン、G2S_GetBG3ControlDCBmp関数ではサブ2Dエンジンの設定を取得できます。

GXBg23ControlDCBmpの定義は以下のようになっております。

typedef union
{
    u16 raw;
    struct
    {
        u16 priority    :2;
        u16 _reserve1   :4;
        u16 mosaic      :1;
        u16 _reserve2   :1;
        u16 screenBase  :5;
        u16 areaOver    :1;
        u16 screenSize  :2;
    };
}
GXBg23ControlDCBmp;

参照

G2_SetBG3ControlDCBmp, G2S_SetBG3ControlDCBmp
