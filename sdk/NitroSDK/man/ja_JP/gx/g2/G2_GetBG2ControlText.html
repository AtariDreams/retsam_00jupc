G2*_GetBG2ControlText

構文

#include <nitro/gx/gx_bgcnt.h>
static inline GXBg23ControlText G2_GetBG2ControlText( void );
static inline GXBg23ControlText G2S_GetBG2ControlText( void );

引数

なし。

返り値

BG2の設定。

説明

テキストBGの場合のBG2の設定をGXBg23ControlText構造体として取得します。
各メンバについてはG2_SetBG2ControlTextを参照してください。

この関数はBG2がテキストBGに設定されている状態で使用する必要があります。

G2_GetBG2ControlText関数ではメイン2Dエンジン、G2S_GetBG2ControlText関数ではサブ2Dエンジンの設定を取得できます。

GXBg23ControlTextの定義は以下のようになっております。

typedef union
{
    u16 raw;
    struct
    {
        u16 priority    :2;
        u16 charBase    :4;
        u16 mosaic      :1;
        u16 colorMode   :1;
        u16 screenBase  :5;
        u16 _reserve    :1;
        u16 screenSize  :2;
    };
}
GXBg23ControlText;

参照

G2_SetBG2ControlText, G2S_SetBG2ControlText
