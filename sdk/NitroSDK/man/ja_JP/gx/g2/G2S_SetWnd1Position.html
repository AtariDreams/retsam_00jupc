G2S_SetWnd1Position

構文

#include <nitro/gx/g2.h>

void G2S_SetWnd1Position(int x1, int y1, int x2, int y2);

引数

x1 ウィンドウ左上X座標(下位8bit有効)
y1 ウィンドウ左上Y座標(0〜192)
x2 ウィンドウ右下X座標(下位8bit有効)
y2 ウィンドウ右下Y座標(0〜192)

返り値

なし。

説明

サブ２Dエンジンのウィンドウ1の位置を設定します。X座標は下位8ビットが有効で、Y座標は0から192までの値である必要があります。
