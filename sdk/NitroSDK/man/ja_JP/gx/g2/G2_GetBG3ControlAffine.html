G2*_GetBG3ControlAffine

構文

#include <nitro/gx/gx_bgcnt.h>
static inline GXBg23ControlAffine G2_GetBG3ControlAffine( void );
static inline GXBg23ControlAffine G2S_GetBG3ControlAffine( void );

引数

なし。

返り値

BG3の設定。

説明

アフィンBGの場合のBG3の設定をGXBg23ControlAffine構造体として取得します。
各メンバについてはG2_SetBG3ControlAffineを参照してください。

この関数はBG3がアフィンBGに設定されている状態で使用する必要があります。

G2_GetBG3ControlAffine関数ではメイン2Dエンジン、G2S_GetBG3ControlAffine関数ではサブ2Dエンジンの設定を取得できます。

GXBg23ControlAffineの定義は以下のようになっております。

typedef union
{
    u16 raw;
    struct
    {
        u16 priority    :2;
        u16 charBase    :4;
        u16 mosaic      :1;
        u16 _reserve    :1;
        u16 screenBase  :5;
        u16 areaOver    :1;
        u16 screenSize  :2;
    };
}
GXBg23ControlAffine;

参照

G2_SetBG3ControlAffine, G2S_SetBG3ControlAffine
