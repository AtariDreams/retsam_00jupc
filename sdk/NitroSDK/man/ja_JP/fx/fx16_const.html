fx16型の定数マクロ

説明

FX16_ONE 1です。 FX16_CONST( 1.0f ) と同等です。
FX16_HALF 1/2です。 FX16_CONST( 0.5f ) と同等です。
FX16_SQRT2 2の平方根です。
FX16_SQRT1_2 1/2の平方根です。
FX16_SQRT3 3の平方根です。
FX16_SQRT1_3 1/3の平方根です。

マクロの定義

NitroSDK/include/nitro/fx/fx_const.hに定義されています。 

#define FX16_ONE                         ((fx16) 0x1000)     // 1.000
#define FX16_HALF                        ((fx16) 0x0800)     // 0.500
#define FX16_SQRT2                       ((fx16) 0x16a1)     // 1.414
#define FX16_SQRT1_2                     ((fx16) 0x0b50)     // 0.707
#define FX16_SQRT3                       ((fx16) 0x1bb6)     // 1.732
#define FX16_SQRT1_3                     ((fx16) 0x093d)     // 0.577
