VEC_Fx16CrossProduct

構文

#include <nitro/fx/fx_vec.h>

void VEC_Fx16CrossProduct(
    const VecFx16 * a,
    const VecFx16 * b,
    VecFx16 * axb
);

引数

a ベクトルへのポインタ
b ベクトルへのポインタ
axb ベクトルaとベクトルbの外積が格納されるポインタ

返り値

なし

説明

VecFx16型ベクトルの外積を求めます。結果は*abに格納されます。
*a,*b,*abは同じベクトルへのポインタでも構いません。

参照

VecFx16, VEC_Fx16DotProduct
