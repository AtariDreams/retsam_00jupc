振動カートリッジ(VIB) API 機能一覧

関数

VIB_Init 振動カートリッジライブラリの初期化を行います。
VIB_End 振動カートリッジライブラリの使用を終了します。
VIB_SetCartridgePulloutCallback 振動カートリッジの抜けコールバックを登録します。
VIB_StartPulse パルス振動を開始します。
VIB_StopPulse パルス振動を停止します。
VIB_IsCartridgeEnabled 振動カートリッジの存在を調べます。
VIB_IsExecuting パルス振動を実行中かどうかを返します。

構造体

VIBPulseState パルス振動のステータス

定数

VIB_PULSE_NUM_MAX パルスセット内のパルスの最大数を得ます。
VIB_ON_TIME_MAX 一回のON時間の最大値を得ます。
VIB_REST_TIME_MIN 休止時間の最小値を得ます。
