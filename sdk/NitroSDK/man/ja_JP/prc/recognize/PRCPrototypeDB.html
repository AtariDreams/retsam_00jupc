PRCPrototypeDB

定義

#include <nitro/prc.h>

typedef struct PRCPrototypeDB
{
    ...
} PRCPrototypeDB;

要素

各要素には直接アクセスしないでください。

説明

図形認識で用いるための見本DBです。各見本パターンの点列データのみ保持する PRCPrototypeList 構造体を元に、各見本パターンを構成する線分の長さや角度、バウンディングボックスなどを事前計算した結果がこの構造体です。

参照

PRC_InitPrototypeDB*, PRC_GetRecognizedEntry*

履歴

2004/06/23 初版
