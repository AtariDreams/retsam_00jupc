PRC_GetInputPatternBufferSize

構文

#include <nitro/prc.h>

u32 PRC_GetInputPatternBufferSize(
    int                         maxPointCount,
    int                         maxStrokeCount );

引数

maxPointCount 加工後の最大点数
maxStrokeCount 加工後の最大画数

返り値

入力パターンデータを作成するのに必要なワークエリアのメモリサイズを返します。

説明

生の入力点列データは、認識で使うPRCInputPattern型の入力パターンデータに展開する必要がありますが、その際に必要なワークエリアのメモリサイズを返します。この返り値以上のサイズのメモリを確保した上で、PRC_InitInputPatternを呼び出してください。

maxPointCount と maxStrokeCount はそれぞれ点数と画数のリサンプル後の最大数を制限するためのパラメータです。

参照

PRC_InitInputPattern, PRC_InitInputPatternEx
