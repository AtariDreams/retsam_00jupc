STD_TSScanf

構文

#include <nitro/std.h>

int STD_TSScanf( const char *src, const char *fmt, ... );

引数

src 入力文字列
fmt 入力書式文字列(フォーマット文字列)

返り値

書式変換が成功し実際に代入された回数を返します。

1回も書式変換が成功せず src の終端に達した場合は -1 を返します。

説明

軽量版の書式文字列関数です。

可変引数を受ける以外はSTD_TVSScanf関数と等価です。
