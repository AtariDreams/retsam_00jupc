STDResult

定義

#include <nitro/std.h>

typedef enum STDResult
{
STD_RESULT_SUCCESS,
STD_RESULT_ERROR,
STD_RESULT_INVALID_PARAM,
STD_RESULT_CONVERSION_FAILED
}
STDResult;


説明

STDライブラリ関数の処理結果を表す列挙定数を示します。
各値の意味は以下の通りです。
値 意味
STD_RESULT_SUCCESS 処理が正しく完了したことを表します。
STD_RESULT_INVALID_PARAM 関数に与えられた引数が異常なために処理が行われなかったことを表します。
STD_RESULT_CONVERSION_FAILED 文字コード変換関数が文字コード変換に失敗して処理を終了したことを表します。
STD_RESULT_ERROR 上記いずれにも該当しないその他の異常で処理が失敗したことを表します。
