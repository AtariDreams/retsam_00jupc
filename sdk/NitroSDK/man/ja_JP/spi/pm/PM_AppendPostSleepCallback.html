PM_AppendPostSleepCallback

構文

#include <nitro/spi.h>
void PM_AppendPostSleepCallback( PMSleepCallbackInfo* info );

引数

info 追加するコールバックの情報を含んだ構造体へのポインタ

返り値

なし。

説明

スリープモードから復帰する際のコールバックを登録します。
コールバックリストの先頭に登録されます。

コールバックはそれぞれ複数登録することが可能です。コールバックは void* 型引数を1つ取るvoid型関数で

typedef void (*PMSleepCallback)( void* );

で定義されています。このコールバック関数は PM_GoSleepMode() の中から呼び出されます。

以下、詳細は PM_AppendPreSleepCallback() のリファレンスを参照して下さい。
