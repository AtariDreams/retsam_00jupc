TP_GetLatestCalibratedPointInAuto

構文

#include <nitro/spi.h>
void TP_GetLatestCalibratedPointInAuto( TPData * result );

引数

result 結果を取得するためのポインタ

返り値

なし。

説明

過去１フレームのオートサンプリング結果から、有効な最新のタッチパネル値を選んで取得することができます。
過去１フレームの中で、有効なサンプリング値が発見されなかった場合には、validityのフラグが立った座標が取得されます。

値のx,y座標はキャリブレーションされ画面座標の範囲の値をとります。（x座標:0〜255、y座標:0〜191）

参照

TPData、TP_GetLatestRawPointInAuto、TP_RequestAutoSamplingStart
