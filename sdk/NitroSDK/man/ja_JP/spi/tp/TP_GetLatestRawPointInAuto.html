TP_GetLatestRawPointInAuto

構文

#include <nitro/spi.h>
void TP_GetLatestRawPointInAuto( TPData * result );

引数

result 結果を取得するためのポインタ

返り値

なし。

説明

過去１フレーム内のオートサンプリング結果の中から、有効な最新のタッチパネル値を選んで取得することができます。
過去１フレームの中で、有効なサンプリング値が発見されなかった場合には、validityのフラグを立てた座標が取得されます。

値のx,y座標はキャリブレーションされていない(0〜4095)の範囲をとります。

参照

TPData、TP_GetLatestCalibratedPointInAuto、TP_RequestAutoSamplingStart
