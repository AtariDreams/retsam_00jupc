WMKeySet

定義

#include <nitro/wm.h>

typedef struct WMKeySet
{
    u16 seqNum;
    u16 rsv;
    u16 key[16];

} WMKeySet;

要素

seqNum キーセットのシーケンシャル番号です。
キーシェア通信に１回成功し接続機器間でキー情報を確実に共有できた都度+1されます。
key 自分自身も含めてシェアされるキー入力情報の配列です。
indexが0のデータが親機のキー情報、indexが1〜15のデータが各AIDの子機のキー情報になります。

説明

WMのキーシェアリング機能により取得するシェアされたキー情報を示す構造体です。
