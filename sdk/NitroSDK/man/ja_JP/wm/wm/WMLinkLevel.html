WMLinkLevel

定義

#include <nitro/wm.h>
typedef enum WMLinkLevel
{ 
    WM_LINK_LEVEL_0 = 0 ,
    WM_LINK_LEVEL_1 ,
    WM_LINK_LEVEL_2 ,
    WM_LINK_LEVEL_3

} WMLinkLevel;

要素

WM_LINK_LEVEL_0 リンク強度 0 です。電波状態が著しく悪い場合、もしくはリンクしていない状態であることを示します。
WM_LINK_LEVEL_1 リンク強度 1 です。電波状態が比較的悪いことを示します。
WM_LINK_LEVEL_2 リンク強度 2 です。電波状態が比較的良好であることを示します。
WM_LINK_LEVEL_3 リンク強度 3 です。電波状態が良好であることを示します。

説明

無線通信時のリンク強度を示す列挙型です。 リンク強度は 0 〜 3 の4段階で表されます。 0 が最低強度もしくはリンクしていない状態、3 が最高強度で良好な電波状態であることを示します。
