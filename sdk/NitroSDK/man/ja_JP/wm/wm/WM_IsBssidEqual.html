WM_IsBssidEqual

構文

#include <nitro/wm/ARM9/wm_api.h>

static inline 
BOOL WM_IsBssidEqual( const u8* idp1 , const u8* idp2 );

static inline
BOOL WM_IsBssidEqual16( const u8* idp1 , const u8* idp2 );

引数

idp1 比較するBSSIDへのポインタ
WM_IsBssidEqual16 では、このアドレスは16bit境界でアラインメントされている必要があります。
idp2 比較するBSSIDへのポインタ
WM_IsBssidEqual16 では、このアドレスは16bit境界でアラインメントされている必要があります。

返り値

BSSIDが同一ならばTRUE
同一でなければFALSEを返します。

説明

２つのBSSIDを比較します。
双方の比較アドレスが16bit境界でアラインメントされている場合に限り WM_IsBssidEqual16 が使用可能で、いくらかのパフォーマンス向上が得られます。
