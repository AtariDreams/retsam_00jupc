WBT_InitParent

構文

#include <nitro/wbt.h>

void WBT_InitParent(int send_packet_size, int recv_packet_size, WBTCallback callback);

引数

send_packet_size WBTのために使用することのできるMP通信の親機パケットサイズ
この値はWBT_PACKET_SIZE_MINバイト以上である必要があります。
recv_packet_size WBTのために使用することのできるMP通信の子機パケットサイズ
この値はWBT_PACKET_SIZE_MINバイト以上である必要があります。
callback 特定のコマンドとは別に発生するイベントを通知するWBTCallback形式の関数
この関数には、WBT_CMD_SYSTEM_CALLBACKコマンドとしてコールバックが通知されます。
このコールバックはWBT_MpParentRecvHook関数の内部から呼び出されます。

返り値

なし

説明

WBT ライブラリを親機として初期化します。
この関数は無線の親機でのみ使用することができます。

無線ライブラリの状態など、この関数の呼び出しに関する条件は特にありません。
