WBTPacketBitmapTable

定義

#include <nitro/wbt.h>

typedef struct {
  u32 *packet_bitmap[WBT_NUM_OF_AID];
} WBTPacketBitmapTable;

説明

各相手局ごとに要求しているデータ受信の進捗管理(受信履歴記録)用ビットマップバッファへのポインタを管理するテーブルです。
これらはWBT_GetBlock関数で相手から受信する際に、
該当するaidのインデックス要素にビットマップバッファへのポインタを指定しておく必要があります。
