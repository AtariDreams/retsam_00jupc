WBTBlockInfoList

定義

#include <nitro/wbt.h>

typedef struct WBTBlockInfoList {
  WBTBlockInfo data_info;
  struct WBTBlockInfoList *next;
  void *data_ptr;
  WBTAidBitmap permission_bmp;
  u16 block_type;
} WBTBlockInfoList;

説明

登録されているブロックリストの情報を表す構造体です。
これらはWBT_RegisterBlock関数でデータブロックを登録する数だけ
ユーザ側で確保しておく必要があります。

この構造体の内部はWBTライブラリで管理されますので、ユーザ側でこれらを操作する必要はありません。
