WBTRequestSyncCallback

定義

#include <nitro/wbt.h>

typedef	struct
{
	WBTBlockNumEntry num_of_list;
	s16 peer_packet_size;
	s16 my_packet_size;
	u16 pad1;
	u32 padd2[2];
} WBTRequestSyncCallback;

説明

WBT_RequestSync関数の完了コールバックで渡される構造体です。

各メンバの内容は以下の通りです。
num_of_list 相手側に登録されているデータブロックの総数です。
peer_packet_size 相手から送信(自身が受信)するMP通信パケットサイズです。
my_packet_size 自身から送信(相手が受信)するMP通信パケットサイズです。
