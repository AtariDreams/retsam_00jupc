CTRDG_IsExisting

構文

#include <nitro/ctrdg.h>
BOOL CTRDG_IsExisting( void );

引数

なし。

返り値

カートリッジが存在していれば TRUE を、そうでない場合は FALSE となります。
但し、一度カートリッジ抜けを検出した後は常に FALSE を返します。

説明

カートリッジが差さっているかどうか調べます。

カートリッジバスがロックされている間は ARM7 による自動抜け検出処理が行われなくなりますので、
ゲームフレーム毎などのプレイヤーが再挿入できない間隔でこの関数を定期的に呼び出して下さい。

参照

CTRDG_Init, CTRDG_IsPulledOut
