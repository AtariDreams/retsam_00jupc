CTRDG_Write*

構文

#include <nitro/ctrdg.h>
BOOL CTRDG_Write8( u8* address, u8 data );
BOOL CTRDG_Write16( u16* address, u16 data );
BOOL CTRDG_Write32( u32* address, u32 data );

引数

address 書き込みアドレス
data 書き込むデータ

返り値

書き込みに成功すればTRUEを返します。カートリッジが存在せず書き込み出来なかった場合は FALSE を返します。

説明

カートリッジ領域に対し書き込みを行います。

CTRDG_Write8() は 8 ビットデータを書き込みます。
CTRDG_Write16() は 16 ビットデータを書き込みます。
CTRDG_Write32() は 32 ビットデータを書き込みます。

アクセスサイクルの設定は行いません。事前に行っておいてください。

参照

CTRDG_Init, CTRDG_DmaCopy*, CTRDG_CpuCopy*, CTRDG_Read*
