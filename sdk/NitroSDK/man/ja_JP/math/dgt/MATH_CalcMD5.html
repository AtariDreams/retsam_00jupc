MATH_CalcMD5

構文

#include <nitro/math/dgt.h>

static inline void MATH_CalcMD5( void* digest, const void* data, u32 dataLength );

引数

digest 計算した MD5 値を格納するメモリへのポインタ。
data 入力データへのポインタ。
dataLength 入力データのサイズ。

返り値

なし。

説明

この関数は、MD5 の計算を行います。
生成される MD5 値の長さは 128 bit、すなわち MATH_MD5_DIGEST_SIZE (= 16) バイトです。このサイズが格納できる領域を digest 引数に渡すようにしてください。
入力データには任意のサイズ・任意のアラインメント位置のものが使用できます。

MATH_MD5Init, MATH_MD5Update, MATH_MD5GetHash の各関数を連続して呼び出すユーティリティ関数です。 内部でスタックに MATHMD5Context 構造体を確保しますので、スタックに十分な余裕がある状態で呼び出してください。

MD5 はメッセージダイジェスト値を求めるためのアルゴリズムです。詳細は IETF (The Internet Engineering Task Force) が発行している RFC (Request For Comments) 1321 を参照してください。
