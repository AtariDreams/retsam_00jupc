MBFakeScanCallback

定義

#include <nitro/mb/mb_fake_child.h>

typedef struct 
{

    u16        index; 
    u16        padding;
    MBGameInfo *gameInfo;
    WMBssDesc  *bssDesc;
} MBFakeScanCallback;

引数

index 取得した親機の管理用インデックス。 MB_FakeEntryToParent関数ではこのインデックスを指定して親機へ接続します。
gameInfo 親機ゲーム情報へのポインタ。
bssDesc 親機の接続情報へのポインタ。

説明

マルチブート親機情報取得時、喪失時にコールバックから得られる親機情報の構造体です。

参照

MB_FakeStartScanParent、MBFakeScanCallbackFunc、MBGameInfo
