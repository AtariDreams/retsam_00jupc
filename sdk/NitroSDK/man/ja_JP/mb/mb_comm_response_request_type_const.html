MBCommResponseRequestType

定義

#include <nitro/mb.h>

typedef	enum
{
    MB_COMM_RESPONSE_REQUEST_KICK,
    MB_COMM_RESPONSE_REQUEST_ACCEPT,
    MB_COMM_RESPONSE_REQUEST_DOWNLOAD,
    MB_COMM_RESPONSE_REQUEST_BOOT
} MBCommResponseRequestType;


説明

マルチブート親機から子機へ通知する接続応答を表す列挙定数を示します。
各値の意味は以下の通りです。
値 意味
MB_COMM_RESPONSE_REQUEST_KICK 子機の接続を拒否して切断させることを表します。
MB_COMM_RESPONSE_REQUEST_ACCEPT 子機の接続を許可してエントリー要求を受け入れることを表します。
MB_COMM_RESPONSE_REQUEST_DOWNLOAD 子機のダウンロード要求を許可してダウンロード開始を通知することを表します。
MB_CommStartSending関数はこの定数を指定してMB_CommResponseRequest関数を呼び出すことと等価です。
MB_COMM_RESPONSE_REQUEST_BOOT 子機のブート要求を許可してブート開始を通知することを表します。
MB_CommBootRequest関数はこの定数を指定してMB_CommResponseRequest関数を呼び出すことと等価です。
