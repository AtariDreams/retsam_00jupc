MB_GetMultiBootDownloadParameter

構文

#include <nitro/mb.h>

static inline const u8* MB_GetMultiBootDownloadParameter(void);

引数

なし

返り値

マルチブート子機として起動した状態であればダウンロード時に指定されたユーザ定義の拡張パラメータを、そうでないならNULLを返します。

説明

プログラム自身がマルチブート子機として起動されたものであれば、 ダウンロード時に指定されたユーザ定義の拡張パラメータが格納されたポインタを返します。
この拡張パラメータは、ダウンロードする親機側がMBGameRegistry構造体で指定します。
