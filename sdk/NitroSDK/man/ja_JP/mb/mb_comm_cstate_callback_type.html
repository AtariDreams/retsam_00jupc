MBCommCStateCallbackFunc

定義

#include <nitro/mb.h>

typedef void (*MBCommCStateCallbackFunc)(u32 status, void* arg);

説明

子機の状態が変更するごとに呼び出されるコールバック関数の形式を定義します。
引数の意味は以下の通りです。
引数 意味
status MBCommCStateで表されるいずれかの状態を示します。
arg 現状、すべてのコールバックにおいてNULLが渡されます。

参照

MBCommCState, MB_FakeSetCStateCallback
