MBIconInfo

定義

#include <nitro/mb.h>

typedef	struct
{
    u16    palette[ MB_ICON_COLOR_NUM ];
    u16    data[ MB_ICON_DATA_SIZE / 2 ];
} MBIconInfo;	/* 544byte */

説明

ダウンロードファイル情報の一部となるアイコンデータの形式を規定する構造体です。
各メンバの意味は以下の通りです。
メンバ 意味
palette アイコンのパレットカラーが格納された配列です。
最大色数は 16 色で、これは MB_ICON_COLOR_NUM として定義されています。
data アイコンのピクセルデータが格納された配列です。
アイコンのサイズは縦横32ピクセルでビット深度が 4 bit(16色)であることから
配列のサイズは 512 バイトとなり、これは MB_ICON_DATA_SIZE として定義されています。
