MB_CommStartSendingAll

構文

#include <nitro/mb.h>

BOOL MB_CommStartSendingAll(void);

引数

なし

返り値

接続許可を受けている子機が存在していれば TRUE を、そうでないなら FALSE を返します。

説明

現在親機から接続許可を受けている全ての子機に対してダウンロード開始通知を送信します。
接続許可を受けている子機が存在していれば TRUE を返します。
親機通信状態がMB_COMM_PSTATE_REQ_WAIT_TO_SENDである時、子機がダウンロード開始通知を待っていることを示します。
