MB_GetGameEntryBitmap

構文

#include <nitro/mb.h>

u16 MB_GetGameEntryBitmap( const MBGameRegistry *game_reg );

引数

game_reg AIDビットマップを取得するMBGameRegistry構造体へのポインタ

返り値

ゲームが配信中の場合はエントリーしているAIDビットマップ(親機自身のAID:0も含む)を、
ゲームが配信中ではない場合は0を返します。

説明

指定したゲーム情報に対して現在エントリーしている全AIDビットマップを取得します。
