MB_CommResponseRequest

構文

#include <nitro/mb.h>

BOOL MB_CommResponseRequest(u16 child_aid, MBCommResponseRequestType ack)

引数

child_aid 応答を送信する子機を指す aid
ack 子機への応答結果を指定するMBCommResponseRequestType型の列挙値

返り値

指定した子機が接続応答待ち状態であり指定した応答種類が正当なものであれば TRUE を、
そうでない場合は FALSE を返します。

説明

指定した子機に対して接続応答を送信します。

親機はゲームの最大接続人数や各子機の接続時情報から任意に判断し、
この関数によって接続の許可および拒否を子機へ通知する必要があります。
親機通信状態コールバックがMB_COMM_PSTATE_REQUESTEDである時、子機がこの応答通知を待っていることを示します。
