MB_CommBootRequest

構文

#include <nitro/mb.h>

BOOL MB_CommBootRequest( u16 child_aid );

引数

child_aid ブート要求を送信する子機を指す aid

返り値

指定した子機に正しくブート要求を送れれば TRUE を、そうでないなら FALSE を返します。

説明

現在親機とマルチブート処理を行っている特定の子機に対してブート要求を送信します。
指定した子機からすでにブート可能状態の通知を受信済みで、
かつその子機へ正しく要求を送信できれば TRUE を返します。
