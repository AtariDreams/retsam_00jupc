MB_CommBootRequestAll

構文

#include <nitro/mb.h>

BOOL MB_CommBootRequestAll(void);

引数

なし

返り値

正しく全ての子機にブート要求を送れれば TRUE を、そうでないなら FALSE を返します。

説明

現在親機とマルチブート処理を行っている全ての子機に対してブート要求を送信します。
すでに全ての子機からブート可能状態の通知を受信済みで、
かつその全ての子機へ正しく要求を送信できれば TRUE を返します。
