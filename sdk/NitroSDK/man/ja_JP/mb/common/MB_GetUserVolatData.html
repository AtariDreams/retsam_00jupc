MB_GetUserVolatData

構文

#include <nitro/mb/mb_gameinfo.h>

void* 
MB_GetUserVolatData( const WMGameInfo* gameInfo );

引数

gameInfo WM_StartScanで取得したビーコン情報へのポインタ。

返り値

MB_SetUserVolatDataで設定したユーザデータへのポインタを取得します。
ユーザデータが含まれないビーコンの場合にはNULLが返ります。

説明

MB親機のビーコンから、MB_SetUserVolatDataで設定されたユーザデータを抽出します。
ユーザデータはMB親機のすべてのビーコンに含まれている訳ではなく、ユーザデータが含まれていないビーコンであった場合にはNULLが返されます。
