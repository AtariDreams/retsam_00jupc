MB_GetMultiBootParentBssDesc

構文

#include <nitro/mb.h>

static inline const MBParentBssDesc* MB_GetMultiBootParentBssDesc(void);

引数

なし

返り値

マルチブート子機として起動した状態であれば有効な親機情報へのポインタを、そうでないならNULLを返します。

説明

プログラム自身がマルチブート子機として起動されたものであれば、有効な親機情報が格納されたポインタを返します。

ただし、ブート直前からこの関数呼び出しまでの環境の何らかの変化によっては、
この関数の返す親機情報がもはや有効なものではなくなっている可能性があることにご注意ください。
