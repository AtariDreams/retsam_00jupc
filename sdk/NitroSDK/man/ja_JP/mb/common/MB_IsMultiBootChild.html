MB_IsMultiBootChild

構文

#include <nitro/mb.h>

static inline BOOL MB_IsMultiBootChild(void);

引数

なし

返り値

マルチブート子機として起動した状態であればTRUEを、そうでないならFALSEを返します。

説明

プログラム自身がマルチブート子機として起動されたものであるか判定します。
この結果がTRUEであれば、MB_GetMultiBootParentBssDesc関数は有効な親機情報を格納しています。

ただし、ブート直前からこの関数呼び出しまでの環境の何らかの変化によっては、
親機情報がもはや有効なものではなくなっている可能性があることにご注意ください。
