CARD_IdentifyBackup

構文

#include <nitro/card.h>

BOOL CARD_IdentifyBackup(CARDBackupType type);

引数

type NITRO-CARDに搭載されているバックアップデバイスの種類
CARD_BACKUP_TYPE_NOT_USEを指定することはできません

返り値

処理が正しく成功すれば TRUE を、そうでないなら FALSE を返します。

説明

NITRO-CARD に搭載されているバックアップデバイスを指定します。
デバイスの指定とともに先頭1バイトの読み出し処理を試行し、成功すれば TRUE を、そうでないなら FALSE を返します。
成否に関するCARDResult型の詳細な結果値はCARD_GetResultCode関数で取得することができます。

この指定以降、CARD 関数は該当デバイスが確かに搭載されている前提のもとに動作します。
実際に搭載されていないデバイスを指定してアクセスすることはプログラムの異常動作および搭載デバイスの破損につながりますので、絶対に避けてください。

この関数は内部でPXIを介してARM7プロセッサと通信しています。
PXIの割り込みが有効でないと処理が完了しない点に注意してください。
NITRO-SDK2.1PR 以降、この関数はカードのテスト読み込みを行う内部処理が追加されました。
この関数を使用する場合、事前にアクセスバスをCARD_LockBackup関数でロックしている必要があります。
