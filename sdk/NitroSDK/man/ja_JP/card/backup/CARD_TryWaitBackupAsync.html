CARD_TryWaitBackupAsync

構文

#include <nitro/card.h>

BOOL CARD_TryWaitBackupAsync(void);

引数

なし

返り値

CARD バックアップデバイスへの非同期関数が呼ばれつつ処理が未完の場合は FALSE を、そうでなければ TRUE を返します。

説明

CARD バックアップデバイスへの非同期処理が完了しているか判定します。
判定の結果にかかわらず、関数はすぐに制御を返します。

この関数が返す値は非同期処理が完了したかどうかに関する真理値です。実際の処理結果を取得するにはCARD_GetResultCode 関数を使用してください。
