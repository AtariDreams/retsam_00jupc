CARD_ReadRom

構文

#include <nitro/card.h>

void CARD_ReadRom(u32 dma, const void *src, void *dst, u32 len);

引数

dma 転送に使用するDMAチャンネル
範囲外のチャンネルに指定するとCPU転送によって実行されます。
有効なDMAチャンネルを指定した場合、転送が完了するまでこのDMAチャンネルを使用できません。
また、カードアクセスの転送元はIOレジスタなのでDMAチャンネル0を指定することはできません。
src 転送元の CARD ROM アドレス
dst 転送先のメモリアドレス
len 読み出すデータのバイト長

返り値

なし

説明

ROM の配置されたデバイスからデータを同期的に読み込みます。
この関数を使用する場合、事前にアクセスバスをロックしている必要があります。

この関数は、CARD_ReadRomAsync関数をコールバックの指定なしで呼び出して
その完了を無条件に待つことと等価です。

関数の内部動作に関してはCARD_ReadRomAsync関数の説明を参照ください。

注意

この関数で有効なDMAチャンネルを指定した場合、転送が完了するまでこのDMAチャンネルを使用できません。
また、この関数は内部で割り込みを使用しますので、割り込みを禁止にしたまま待ち続けると処理が完了しないという点に注意してください。
