WFSPacketBuffer

定義

#include <nitro/wfs.h>

typedef struct WFSPacketBuffer
{
    u8     *buffer;         /* 送信用バッファまたは受信バッファ */
    int     length;         /* 送信可能最大サイズまたは受信サイズ */
    int     bitmap;         /* AID ビットマップ */
}
WFSPacketBuffer;

要素

buffer 送信用バッファまたは受信バッファ
length 送信可能最大サイズまたは受信サイズ
bitmap 送信元または送信先を示すビットマップ

説明

WFSライブラリのMP通信パケット送受信フック関数で使用するパケット情報をあらわす構造体です。

参照

WFS_CallServerPacketSendHook WFS_CallServerPacketRecvHook WFS_CallClientPacketSendHook WFS_CallClientPacketRecvHook
