WFSRequestClientReadDoneCallback

定義

#include <nitro/wfs.h>

typedef void (*WFSRequestClientReadDoneCallback)(WFSClientContext *context, BOOL succeeded, void *arg);

説明

WFS_RequestClientRead関数で使用される完了コールバックのプロトタイプ宣言です。
各引数の意味は以下の通りです。

要素

context WFSClientContext構造体のポインタ
succeeded リクエストが正常に完了すればTRUE、失敗すればFALSE
リクエストを処理中にWFS_EndClient関数が呼び出された場合に失敗します。
arg WFS_RequestClientRead関数で指定した任意の引数

参照

WFS_RequestClientRead WFS_EndClient
