WFS_CallServerDisconnectHook

構文

#include <nitro/wfs.h>

void WFS_CallServerDisconnectHook(WFSServerContext *context, const WFSPeerInfo *peer);

引数

context WFSServerContext構造体のポインタ
peer 切断した子機の情報が格納されたWFSPeerInfo構造体のポインタ

返り値

なし

説明

子機が切断したことをライブラリへ通知します。
アプリケーションはワイヤレス通信で子機が切断するたびにこの関数を呼び出す必要があります。
