WFSPeerInfo

定義

#include <nitro/wfs.h>

typedef struct WFSPeerInfo
{
    int     aid;            /* AID */
    u8      mac[6];         /* MAC アドレス */
    u8      padding[2];
}
WFSPeerInfo;

要素

aid 通知すべき対象局のAID
mac 通知すべき対象局のMACアドレス

説明

WFSライブラリのMP通信フック関数に使用される引数構造体です。
通信先の接続や切断をWFSライブラリに検知させるため、この構造体に適切な情報を格納して該当するフック関数を呼び出す必要があります。

参照

WFS_CallServerConnectHook WFS_CallServerDisconnectHook WFS_CallClientConnectHook WFS_CallClientDisconnectHook
