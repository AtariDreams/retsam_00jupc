WFSEventType

定義

#include <nitro/wfs.h>

/* イベントコールバック種別 */
typedef enum WFSEventType
{
    /*** 親機にのみ通知されるイベント ***/

    WFS_EVENT_SERVER_SEGMENT_REQUEST, /* 子機からの読み込みリクエスト. (引数はWFSSegmentBuffer*) */

    /*** 子機にのみ通知されるイベント ***/

    WFS_EVENT_CLIENT_READY            /* 親機からのファイルテーブルの受信完了 */
}
WFSEventType;

説明

WFSライブラリのイベントを表す列挙定数を示します。
各値の意味は以下の通りです。

親機にのみ通知されるイベント
値 通知内容 引数の意味
WFS_EVENT_SERVER_SEGMENT_REQUEST 子機からの読み込みリクエストが発生した時点で通知されます。
親機は指定された領域のデータをデバイスから読み込む必要があります。		 リクエスト内容を保持するWFSSegmentBuffer型へのポインタ
子機にのみ通知されるイベント
値 通知内容 引数の意味
WFS_EVENT_CLIENT_READY 親機からファイルテーブルの受信が完了した時点で通知されます。
この通知以降、WFS_GetTableFormat関数でファイルテーブルを参照可能になります。		 引数はありません。
