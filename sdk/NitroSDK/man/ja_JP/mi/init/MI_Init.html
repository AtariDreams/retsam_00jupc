MI_Init

構文

#include <nitro/mi.h>

void MI_Init( void );

引数

なし。

返り値

なし。

説明

MI 関連の初期化を行ないます。

この関数は、OS_Init() を使用している場合、そこから呼ばれますのでアプリケーションが改めて呼ぶ必要はありません。(ARM9 の場合。ARM7 では OS_Init() から呼んでいません。)

具体的には以下のことを行なっています。

ARM9
共用の WRAM を全て ARM7 に割り当てます。( MI_SetWramBank( MI_WRAM_ARM7_ALL ) )
DMA0 を確実に初期化します。これはDMAハードウェア不具合への対処です。( MI_StopDma( 0 ) )

ARM7
DMA0 を確実に初期化します。これはDMAハードウェア不具合への対処です。( MI_StopDma( 0 ) )

参照

MI_SetWramBank, MI_StopDma
