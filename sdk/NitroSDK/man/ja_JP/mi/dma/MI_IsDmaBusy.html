MI_IsDmaBusy

構文

#include <nitro/mi.h>

BOOL MI_IsDmaBusy( u32 dmaNo )

引数

dmaNo 使用DMAチャンネル

返り値

指定DMAチャンネルが使用中かどうかを返します。

説明

DMAチャンネルか使用中かどうかを取得します。
返り値が TRUE の場合、まだbusyであることを示します。

内部動作

IOレジスタのDMAnコントロールレジスタ(0x40000B8 + 12n)を使用します。( n は 使用するDMAチャンネルです。)

参照

MI_DmaCopy*, MI_WaitDma

履歴
