MI_ReadByte

構文

#include <nitro/mi.h>

u8 MI_ReadByte( const void* address )

引数

address バイトデータを読むアドレス(ポインタ)

返り値

指定アドレスのメモリ内容(u8)

説明

const void型ポインタで指定されたアドレス address の先のメモリ内容をu8データで取得します。
内部ではメモリにアクセスするのはu16データとして行っていますので、アセンブラレベルで strb/ldrb のようなバイトアクセス命令は生成されません。
