MI_SetCartridgeRamCycle

構文

#include <nitro/mi.h>

void MI_SetCartridgeRamCycle( MICartridgeRamCycle c )

引数

c カートリッジRAM領域へのアクセスサイクル

返り値

なし。

説明

カートリッジRAM領域へのアクセスサイクルを設定します。
カートリッジROM領域への設定は、MI_SetCartridgeRomCycle1st() や MI_SetCartfigeRomCycle2nd() で行います。

引数 c は、MICartridgeRamCycle 列挙型で、以下の意味を持ちます。

MI_CTRDG_RAMCYCLE_6 6サイクル
MI_CTRDG_RAMCYCLE_8 8サイクル
MI_CTRDG_RAMCYCLE_10 10サイクル
MI_CTRDG_RAMCYCLE_18 18サイクル

内部動作

IOレジスタの、EXMEMCNT(0x4000204)にアクセスしています。

参照

MI_GetCartridgeRamCycle
MI_SetCartridgeRomCycle1st, MI_SetCartridegeRomCycle2nd
