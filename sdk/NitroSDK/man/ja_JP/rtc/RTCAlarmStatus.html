RTCAlarmStatus

定義

#include <nitro/rtc.h>
typedef enum
{ 
	RTC_ALARM_STATUS_OFF = 0 ,
	RTC_ALARM_STATUS_ON ,
	RTC_ALARM_STATUS_MAX

} RTCAlarmStatus;

要素

RTC_ALARM_STATUS_OFF アラーム割り込みは禁止されている状態
RTC_ALARM_STATUS_ON アラーム割り込みが許可されている状態

説明

リアルタイムクロックに登録しておくことのできるアラームの動作状態です。
アラーム割り込みが許可されている場合は、分の桁上げ処理の度にアラーム設定値と現在値が比較され、一致した場合に割り込みが発生します。
なお、割り込み禁止の状態でアラーム設定値の読み出し、書き込みを行うことはできません。
