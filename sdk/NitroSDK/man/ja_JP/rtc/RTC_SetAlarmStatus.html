RTC_SetAlarmStatus

構文

#include <nitro/rtc.h>
RTCResult RTC_SetAlarmStatus( 
            RTCAlarmChan		chan, 
            const RTCAlarmStatus*	status );

引数

chan アラームチャンネルをRTCAlarmChan列挙型で指定します。
status アラーム状態を格納したバッファへのポインタ

返り値

要求が成功した場合に 0 を返します。
要求が失敗した場合には 0 以外を返します。

説明

リアルタイムクロックにアラーム割り込み許可状態を書き込みます。
この関数はARM7にアラーム割り込み許可状態書き込み要求を発行し、ARM7からの応答を待って処理結果を返します。
RTC_SetAlarmStatusAsyncの同期版です。

注意

書き込むアラーム状態には、許可／禁止 以外の状態を指定しないで下さい。
また、この関数は同期を取る為にPXI割り込みを待ちますので、PXI割り込みを禁止した状態で呼び出さないようにして下さい。
