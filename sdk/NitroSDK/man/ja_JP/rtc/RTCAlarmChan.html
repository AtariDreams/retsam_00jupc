RTCAlarmChan

定義

#include <nitro/rtc.h>
typedef enum
{ 
	RTC_ALARM_CHAN_1 = 0 ,
	RTC_ALARM_CHAN_2 ,
	RTC_ALARM_CHAN_MAX

} RTCAlarmChan;

要素

RTC_ALARM_CHAN_1 チャンネル１
RTC_ALARM_CHAN_2 チャンネル２

説明

リアルタイムクロックに登録しておくことのできるアラームにはチャンネルが２つあります。
各々のチャンネルは独立して動作可能ですが割り込み出力は共通である為、２つのチャンネルで同時に割り込みが発生した場合には割り込みは１回しか発生しません。
