RTCTime

定義

#include <nitro/rtc.h>
typedef struct {
         u32        hour;
         u32        minute;     
         u32        second;
} RTCTime;

要素

hour 時を示す 0〜23 の数値
minute 分を示す 0〜59 の数値
second 秒を示す 0〜59 の数値

説明

リアルタイムクロックの時刻を表す構造体です。
時分秒を示します。
