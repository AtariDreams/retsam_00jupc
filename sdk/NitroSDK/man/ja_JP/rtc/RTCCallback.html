RTCCallback

定義

#include <nitro/rtc.h>
typedef void ( *RTCCallback )( 
            RTCResult	result, 
            void*	arg );

引数

result 非同期処理の結果を示すRTCResult列挙型の値
arg 非同期関数を呼び出す際に指定された引数

返り値

なし。

説明

リアルタイムクロックを非同期的に操作する関数によって起動された非同期処理が完了した際に呼び出されるユーザコールバック関数の形式です。
result引数にはRTCResult列挙型の処理結果が、arg引数には非同期関数を呼び出す際に指定した引数が、それぞれ渡されます。
