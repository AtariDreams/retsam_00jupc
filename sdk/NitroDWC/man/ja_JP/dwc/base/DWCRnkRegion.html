DWCRnkRegion

定義


#include <dwc.h>

typedef enum{

    DWC_RNK_REGION_JP   = 0x01,
    DWC_RNK_REGION_US   = 0x02,
    DWC_RNK_REGION_EU   = 0x04,
    DWC_RNK_REGION_KR   = 0x08,

    DWC_RNK_REGION_ALL  = 0xff

} DWCRnkRegion;

説明

汎用ランキングのリージョンコードを示す列挙型です。

DWC_RnkGetScoreAsync関数の呼び出しでは、複数の値のOR値を指定することができます。

DWC_RNK_REGION_JP 日本
DWC_RNK_REGION_US 米国
DWC_RNK_REGION_EU 欧州
DWC_RNK_REGION_KR 韓国
DWC_RNK_REGION_ALL 全世界
