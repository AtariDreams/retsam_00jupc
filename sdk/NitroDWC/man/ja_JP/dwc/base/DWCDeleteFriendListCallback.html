DWCDeleteFriendListCallback

構文

#include <dwc.h>

typedef void (*DWCDeleteFriendListCallback)( int   deletedIndex,
                                             int   srcIndex,
                                             void* param );

説明

DWC_UpdateServersAsync関数の呼び出し以降に、ローカルの友達リスト中に同じ友達情報を見つけ、ライブラリ側で消去した時に呼び出されます。

本コールバック関数はDWC_UpdateServersAsync関数によって設定することができます。

引数

deletedIndex 削除された友達の友達リストインデックス
srcIndex 同じ友達情報と判定された友達リストインデックス
param DWC_UpdateServersAsync関数で指定したコールバック用パラメータ

返り値

なし。

参照

DWC_UpdateServersAsync
