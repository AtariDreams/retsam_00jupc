DWC_GetMyAID

構文


#include <dwc.h>

u8 DWC_GetMyAID( void );

説明

自分のAIDを取得します。

AIDはメッシュ型ネットワーク内での各DSの識別番号で、0〜（ネットワーク構成台数-1）の数値が用いられます。 例えば4人でマッチメイクを完了した場合なら、0, 1, 2, 3 の4台が存在し、AID = 1 の人がネットワークから抜ければ、残りは 0, 2, 3 となります。
サーバクライアントマッチメイクにおいてはサーバDSのAIDが必ず0になります。

引数

なし

返り値

自分のAID。FriendsMatchライブラリ非動作中に本関数を呼び出した場合は0を返す。
