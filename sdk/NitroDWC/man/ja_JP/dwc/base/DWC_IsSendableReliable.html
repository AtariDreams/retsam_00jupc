DWC_IsSendableReliable

構文

#include <dwc.h>

BOOL DWC_IsSendableReliable( u8 aid );

説明

特定のAIDに割り当てられたホストへ、データをReliable送信可能かをチェックします。

DWC_SendReliable / DWC_SendReliableBitmap関数によるReliable送信は、まだ送信中のデータがある場合、もしくはDWC_InitFriendsMatch関数でサイズを指定した送信バッファに空きが全くない場合は、データを送信できません。その場合は本関数はFALSEを返すので、本関数がTRUEを返すまで、送信しないようにしてください。

またエラー発生中も本関数はFALSEを返します。

なお、引数aidに無効なAIDを指定した場合もFALSEを返しますが、この場合はDWC_IsValidAID関数でAIDを調べれば、前の2つの送信不可条件と区別することができます。

引数

aid 送信相手のAID

返り値

TRUE 送信可能
FALSE 送信不可

参照

DWC_SendReliable, DWC_SendReliableBitmap, DWC_InitFriendsMatch, DWC_IsValidAID
