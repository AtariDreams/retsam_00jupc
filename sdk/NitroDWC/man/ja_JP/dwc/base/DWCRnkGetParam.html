DWCRnkGetParam

定義


#include <dwc.h>

typedef struct{
    u32             size;
    union{
        u32         data;
        struct{
            u32 sort;
            u32 since;
        }           order;
        struct{
            u32 sort;
            u32 limit;
            u32 since;
        }           toplist;
        struct{
            u32 sort;
            u32 limit;
            u32 since;
        }           near;
        struct{
            u32 sort;
            u32 limit;
            u32 since;
            s32 friends[64];
        }           friends;
    };
} DWCRnkGetParam;

説明

DWC_RnkGetScoreAsync関数で指定するパラメータを格納する構造体です。

取得モードに応じて該当するパラメータに値をセットします。
取得モードがDWC_RNK_GET_MODE_ORDERの場合はDWCRnkGetParam.order以下を設定します。
取得モードがDWC_RNK_GET_MODE_TOPLISTの場合はDWCRnkGetParam.toplist以下を設定します。
取得モードがDWC_RNK_GET_MODE_NEARの場合はDWCRnkGetParam.near以下を設定します。
取得モードがDWC_RNK_GET_MODE_FRIENDSの場合はDWCRnkGetParam.friends以下を設定します。
取得モードがDWC_RNK_GET_MODE_NEAR_HIの場合はDWCRnkGetParam.near以下を設定します。
取得モードがDWC_RNK_GET_MODE_NEAR_LOWの場合はDWCRnkGetParam.near以下を設定します。

size 後続のデータサイズ(共通)。
取得モードがDWC_RNK_GET_MODE_ORDERの場合はsizeof(DWCRnkGetParam.order)
取得モードがDWC_RNK_GET_MODE_TOPLISTの場合はsizeof(DWCRnkGetParam.toplist)
取得モードがDWC_RNK_GET_MODE_NEARの場合はsizeof(DWCRnkGetParam.near)
取得モードがDWC_RNK_GET_MODE_FRIENDSの場合はsizeof(DWCRnkGetParam.friends)
取得モードがDWC_RNK_GET_MODE_NEAR_HIの場合はsizeof(DWCRnkGetParam.near)
取得モードがDWC_RNK_GET_MODE_NEAR_LOWの場合はsizeof(DWCRnkGetParam.near)
limit 取得する最大数を指定します。DWC_RNK_GET_MAX(30)以下の値を設定してください。下限値は、取得モードによって異なり、DWC_RNK_GET_MODE_TOPLISTの場合は１、DWC_RNK_GET_MODE_NEAR、DWC_RNK_GET_MODE_FRIENDS、DWC_RNK_GET_MODE_NEAR_HI、DWC_RNK_GET_MODE_NEAR_LOWのときは２となります。
sort 並び順
DWC_RNK_ORDER_ASC:昇順
DWC_RNK_ORDER_DES:降順
since 現時点から指定分前までに更新された中でのランキングを取得します。０を指定した場合はすべてのデータ中でのランキングを取得します。
friends[64] 友達のGSプロファイルIDリスト
64個に満たない場合は、先頭から格納し、余った領域は0で埋めてください。
