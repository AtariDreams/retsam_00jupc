DWC_Base64Decode

構文

#include <dwc.h>

int DWC_Base64Decode(const char *src, const u32 srclen, char *dst, const u32 dstlen);

説明

srcで指定されたBase64データをデコードし、char型のデータとして出力します。

dstにNULLを指定すれば、実際には変換は行わず、予想される変換後の最大データサイズを返り値として返します。最大データサイズは最も変換の効率が悪かった場合の値で、実際に変換した場合はそれよりも小さくなる場合もあります。

本関数は特殊なBase64を使用しています。
本関数でデコードできるのは、DWC_Base64Encode関数を用いてエンコードしたデータになります。

引数

src 変換元Base64データへのポインタ
srclen 変換元Base64データサイズ（文字列長）
dst 変換後データ格納用バッファへのポインタ
dstlen 変換後データ格納用バッファサイズ

返り値

-1 変換失敗。
予想される変換後の最大データサイズが引数dstlenの値を越えているか、引数srclenが4の倍数でありません。
それ以外 変換後データの実際のサイズ

参照
