DWC_CheckHasProfile

構文

#include <dwc.h>

BOOL DWC_CheckHasProfile( const DWCUserData* userdata );

説明

userdataで指定したユーザデータが既にWi-Fiコネクションに接続し、有効なログインID（プロファイル）を含んでいるかどうかをチェックします。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ

返り値

TRUE 有効なログインIDが含まれている
FALSE 有効なログインIDが含まれていない

参照

DWC_CreateUserData, DWC_LoginAsync
