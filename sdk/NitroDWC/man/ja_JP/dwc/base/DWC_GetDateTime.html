DWC_GetDateTime

構文

#include <dwc.h>

BOOL DWC_GetDateTime(RTCDate *date, RTCTime *time);

説明

（認証処理時の認証サーバの時刻情報を元にした）現在の日付と時刻をGMTで取得します。

この関数で日付と時刻を取得するには、あらかじめDWC_LoginAsync関数又はDWC_NASLoginAsync関数でWi-Fiコネクションへの接続を完了している必要があります。

この関数で取得できる日付と時刻は、認証処理時に記録した認証サーバの時刻情報とリアルタイムクロックの情報との差を使って生成しています。

NTPサーバを使って取得できるような正確な情報ではないことにご注意ください。（分秒単位の誤差を考慮してください）

引数

date 現在の日付(GMT)を格納する領域へのポインタ。
time 現在の時刻(GMT)を格納する領域へのポインタ。

返り値

TRUE 日付と時刻の取得に成功した。
FALSE 日付と時刻の取得に失敗した。

参照

DWC_LoginAsync, DWC_NASLoginAsync
