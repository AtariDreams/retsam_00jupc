DWC_Base64Encode

構文

#include <dwc.h>

int DWC_Base64Encode(const char *src, const u32 srclen, char *dst, const u32 dstlen);

説明

srcで指定されたデータをBase64でエンコードし、char型の文字列として出力します。

dstにNULLを指定すれば、実際には変換は行わず、予想される変換後の最大文字列長を返り値として返します。最大文字列長は最も変換の効率が悪かった場合の値で、実際に変換した場合はそれよりも短くなる場合もあります。

変換後の文字列はヌル文字終端を含みません。

本関数は特殊なBase64を使用しています。
本関数でエンコードしたデータのデコードには、必ずDWC_Base64Decode関数を用いてください。

引数

src 変換元データへのポインタ
srclen 変換元データサイズ
dst 変換後データ格納用バッファへのポインタ
dstlen 変換後データ格納用バッファサイズ（文字列長）

返り値

-1 変換失敗。
予想される変換後の最大文字列長が引数dstlenの値を越えています。
それ以外 変換後データの実際のサイズ（文字列長）

参照

DWC_Base64Decode
