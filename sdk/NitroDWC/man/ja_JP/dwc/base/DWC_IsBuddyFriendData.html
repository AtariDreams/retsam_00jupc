DWC_IsBuddyFriendData

構文

#include <dwc.h>

BOOL DWC_IsBuddyFriendData( const DWCFriendData* frienddata );

説明

frienddataで指定された友達情報について、既にGameSpyサーバ上で友達関係が成立しているかを調べます。

GameSpyサーバ上で友達関係が成立すると、友達情報内の友達成立フラグがセットされるため、このフラグをチェックして友達関係の成立／不成立を調べています。
また、友達成立フラグはセットされた後で相手が友達関係を破棄してもリセットされないため、本関数で友達関係が維持されているか（お互いに相手を友達として認めているか）を確認することはできません。

引数

frienddata 友達情報の格納されたDWCFriendData構造体のバッファへのポインタ

返り値

TRUE 既にGameSpyサーバ上で友達関係が成立している。
FALSE まだ友達関係が成立していない。
