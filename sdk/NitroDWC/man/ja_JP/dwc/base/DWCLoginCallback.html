DWCLoginCallback

構文

#include <dwc.h>

typedef void (*DWCLoginCallback)( DWCError error,
                                  int  profileID,
                                  void *param );

説明

DWC_LoginAsync関数を呼び出した後、Wi-Fiコネクションへの接続完了時に呼び出されます。

本コールバック関数はDWC_LoginAsync関数によって設定することができます。

引数

error DWCエラー種別。内容の詳細は、DWC_GetLastErrorExを参照してください。
profileID 取得できた自分のGSプロファイルID
param DWC_LoginAsync関数で指定したコールバック用パラメータ

返り値

なし。

参照

DWC_LoginAsync, DWC_GetLastErrorEx
