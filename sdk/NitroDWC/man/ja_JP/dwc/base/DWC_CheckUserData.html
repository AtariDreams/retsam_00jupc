DWC_CheckUserData

構文

#include <dwc.h>

BOOL DWC_CheckUserData( const DWCUserData* userdata );

説明

userdataに指定されたユーザデータが正当な状態かどうかをチェックします。 ユーザデータ内には、CRCが含まれており、それを用いて正当性をチェックしています。 セーブされたユーザデータをメモリに読み込んだ際のチェックに使用してください。

ユーザデータをセーブする際に、DWC_ClearDirtyFlag関数でDirtyフラグをクリアしていなかった場合、正当ではないと判定されることがあることに注意してください。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ

返り値

TRUE 正当
FALSE 正当ではない

参照

DWC_CreateUserData, DWC_ClearDirtyFlag
