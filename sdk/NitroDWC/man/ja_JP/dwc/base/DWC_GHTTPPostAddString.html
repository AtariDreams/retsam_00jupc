DWC_GHTTPPostAddString

構文

#include <dwc.h>

BOOL DWC_GHTTPPostAddString( DWCGHTTPPost* post,
                             const char* key,
                             const char* value );

説明

HTTPサーバにアップロードするデータをDWCGHTTPPost型オブジェクトに追加します。 DWCGHTTPPost型オブジェクトは、データをHTTPサーバにアップロードするための情報単位で、DWC_GHTTPNewPost関数によって作成されます。

データはkey文字列、value文字列をセットにしてHTTPサーバにアップロードされます。 key／value文字列ともに、コピーしてライブラリ内で保持されます。 また、両文字列ともにヌル文字終端されている必要があります。

実際にデータのアップロードを開始するためには、DWC_PostGHTTPData関数、DWC_GetGHTTPDataEx2関数を使います。

HTTPサーバに送信されるデータは、下記サンプルプログラムの例では以下のような文字列になります。

"key1=value1&key2=value2"

同一DWCGHTTPPost型オブジェクトに対して更にデータが追加されれば、以下のように文字列が追加されていきます。

"key1=value1&key2=value2&key3=value3&key4=value4…" 


DWCGHTTPPost post;

// DWCGHTTPPost型オブジェクトを作成する
DWC_GHTTPNewPost( &post );

// DWCGHTTPPost型オブジェクトにアップロードするデータをセットする
DWC_GHTTPPostAddString( &post, "key1", "value1" );
DWC_GHTTPPostAddString( &post, "key2", "value2" );

引数

post データを追加するDWCGHTTPPost型オブジェクトへのポインタ
key 追加するデータの名前を表す文字列。NULLおよび空文字列""は指定できません。
value 追加するデータ本体の文字列。NULLを指定すると、空文字列""を指定したことになります。

返り値

TRUE データ追加成功
FALSE パラメータエラー、もしくはメモリ確保エラーのためデータ追加失敗

参照

DWC_GHTTPNewPost, DWC_PostGHTTPData, DWC_GetGHTTPDataEx2
