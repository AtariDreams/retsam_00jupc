DWC_LogoutFromStorageServer

構文

#include <dwc.h>

void DWC_LogoutFromStorageServer( void );

説明

データストレージサーバからログアウトします。この関数を呼び出さずにDWC_ShutdownFriendsMatch関数を呼び出すと、自動的にログアウトします。
本関数をコールバック関数DWCStorageLoginCallback、DWCSaveToServerCallback、DWCLoadFromServerCallbackの内部から呼び出さないでください。

引数

なし。

返り値

なし。

参照

DWC_LoginToStorageServerAsync, DWC_ShutdownFriendsMatch, DWCStorageLoginCallback, DWCSaveToServerCallback, DWCLoadFromServerCallback
