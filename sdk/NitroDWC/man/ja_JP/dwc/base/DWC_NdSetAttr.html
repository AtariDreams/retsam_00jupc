DWC_NdSetAttr

構文


#include <nd/dwc_nd.h>

BOOL DWC_NdSetAttr( const char* attr1, const char* attr2, const char* attr3 )

説明

ダウンロードライブラリに対して属性文字列を設定します。属性文字列はそれぞれ10文字以内のヌル文字終端したASCII文字列を指定してください。

属性文字列はダウンロードサーバがダウンロード可能なファイルを選択する際の基準となります。
3つの属性文字列は以下のような方法で評価され、3つの属性文字列の全てでダウンロード可能と評価されたときにそのファイルはダウンロード可能なものとして扱われます。

  • 属性文字列に値が設定されている場合は、その属性文字列がサーバ側で設定された属性文字列と完全一致した場合にダウンロード可能と評価されます。
  • 属性文字列が空文字列の場合は、無条件でダウンロード可能と評価されます。

この関数を呼び出さずにDWC_NdGetFileListNumAsync関数、DWC_NdGetFileListAsync関数、DWC_NdGetFileAsync関数の各関数を呼び出した場合は、全ての属性に空文字列が設定された状態で動作します。

引数

attr1 設定する属性文字列1へのポインタ
attr2 設定する属性文字列2へのポインタ
attr3 設定する属性文字列3へのポインタ

返り値

TRUE 成功
FALSE 失敗

参照

DWC_NdGetFileListNumAsync, DWC_NdGetFileListAsync, DWC_NdGetFileAsync
