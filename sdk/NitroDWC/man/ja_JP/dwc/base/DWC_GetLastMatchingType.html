DWC_GetLastMatchingType

構文

#include <dwc.h>

int DWC_GetLastMatchingType( void );

説明

最後に行ったマッチメイクの種別を取得します。

DWC_InitFriendsMatch関数でFriendsMatchライブラリの初期化を行っていない場合は、-1を返します。 初期化を行った後、一度目のマッチメイクを開始するまでの間は、DWC_MATCH_TYPE_ANYBODY = 0 を返します。

引数

なし

返り値

-1 FriendsMatchライブラリが初期化されていない
DWC_MATCH_TYPE_ANYBODY 友達無指定ピアマッチメイク
DWC_MATCH_TYPE_FRIEND 友達指定ピアマッチメイク
DWC_MATCH_TYPE_SC_SV サーバクライアントマッチメイクのサーバDS
DWC_MATCH_TYPE_SC_CL サーバクライアントマッチメイクのクライアントDS

参照

DWC_InitFriendsMatch, DWC_ConnectToAnybodyAsync, DWC_ConnectToFriendsAsync, DWC_SetupGameServer, DWC_ConnectToGameServerAsync
