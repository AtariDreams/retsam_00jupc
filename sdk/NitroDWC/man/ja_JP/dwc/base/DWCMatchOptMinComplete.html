DWCMatchOptMinComplete

定義


#include <dwc.h>

typedef struct DWCstMatchOptMinComplete
{
    u8  valid;
    u8  minEntry;
    u8  pad[2];
    u32 timeout;
}
DWCMatchOptMinComplete;

説明

指定人数以下マッチメイク完了オプションを設定するための構造体です。

このマッチメイクオプションは、ピアマッチメイクにおいて、一定時間を過ぎてもマッチメイクが完了しない場合に、DWC_ConnectToAnybodyAsync / DWC_ConnectToFriendsAsync関数で指定した人数以下でのマッチメイクを許可するものです。 DWC_SetMatchingOption関数の引数opttypeにDWC_MATCH_OPTION_MIN_COMPLETEを指定し、引数optvalにメンバ変数に値をセットした本構造体へのポインタを指定して呼び出すことによって設定することができます。

自分のマッチメイク開始からの経過時間がtimeoutに達したからといってすぐにマッチメイクを終了するわけではなく、その時に接続している全てのホストが同様にtimeoutの時間を越えていなければマッチメイクが完了することはありません。

valid TRUE :このマッチメイクオプションを有効にする。
FALSE:このマッチメイクオプションを無効にする。
minEntry マッチメイク完了としても良い、自分を含めた最低接続人数。
pad パディング。何もセットしなくても構いません。
timeout 指定人数以下でもマッチメイク完了を許可するようになるまでの、マッチメイク開始からの経過時間（単位：msec）。

DWC_SetMatchingOption, DWC_GetMOMinCompState, DWC_ConnectToAnybodyAsync, DWC_ConnectToFriendsAsync
