DWC_Alloc

構文

#include <dwc.h>

void DWC_Alloc  ( DWCAllocType name, u32 size );
void DWC_AllocEx( DWCAllocType name, u32 size, int align );

説明

DWC_SetMemFunc関数で設定したメモリ確保関数を用いてメモリを確保します。

DWC_Alloc関数の場合、メモリ確保関数の引数alignには32が渡されます。
DWC_AllocEx関数の場合、メモリ確保関数の引数alignには、本関数の引数alignが渡されます。

引数

name ライブラリ開発用の情報です。0を指定してください。
size 確保したいメモリのサイズ（バイト）。
align 確保したいメモリのアライメント（バイト）。DWC_SetMemFunc関数で設定したメモリ確保関数に渡されます。

返り値

DWC_SetMemFunc関数で設定したメモリ確保関数が確保したメモリへのポインタを、そのまま返します。

参照

DWC_SetMemFunc DWC_Realloc, DWC_ReallocEx, DWC_Free
