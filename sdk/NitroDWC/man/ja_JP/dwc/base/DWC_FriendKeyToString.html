DWC_FriendKeyToString

構文

#include <dwc.h>

void DWC_FriendKeyToString( char* string, u64 friendKey );

説明

友達登録鍵friendKeyを10進12桁の文字列に変換し、引数stringで指定された領域に終端文字 '\0' 付きで書き込みます。

引数stringで指定された領域は、DWC_FRIENDKEY_STRING_BUFSIZEバイトの長さが必要です。

引数

string 10進12桁の文字列に変換された友達登録鍵の書き込み先へのポインタ
friendKey 文字列に変換する友達登録鍵

返り値

なし

参照

DWC_StringToFriendKey
