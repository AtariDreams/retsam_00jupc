DWC_ReportFriendData

構文

#include <dwc.h>

void DWC_ReportFriendData( const DWCUserData* userdata,
                           const DWCFriendData* frienddata );

説明

frienddataで指定された友達情報を、OS_TPrintfを用いて表示します。（デバッグ用）

これはデバッグ用の関数ですが、最終ROM版（FINALROM）ライブラリでも動作します。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ
frienddata 友達情報の格納されたDWCFriendData構造体のバッファへのポインタ

返り値

なし
