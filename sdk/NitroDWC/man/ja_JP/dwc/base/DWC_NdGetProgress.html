DWC_NdGetProgress

構文


#include <nd/dwc_nd.h>

BOOL DWC_NdGetProgress( u32* received, u32* contentlen )

説明

実行中のファイルダウンロードの進行状況を取得します。

現在ダウンロード中のファイルの総容量が不明な場合は、contentlenで指定された領域には0が書き込まれます。

引数

received ダウンロード済みバイト数を格納する領域へのポインタ
contentlen ファイルの総容量を格納する領域へのポインタ

返り値

TRUE 成功
FALSE 失敗、あるいはダウンロード処理が実行中でない

参照

なし
