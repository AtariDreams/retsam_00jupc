DWC_ProcessGHTTP

構文

#include <dwc.h>

BOOL DWC_ProcessGHTTP( void );

説明

GHTTPライブラリでの通信処理を実行します。 毎ゲームフレーム程度の頻度で呼び出してください。

この関数を呼び出さないと DWC_GetGHTTPData 関数、 DWC_GetGHTTPDataEx 関数、 DWC_GetGHTTPDataEx2 関数、 DWC_PostGHTTPData 関数 に必要な処理をさせることが出来ないので、必ず呼び出すようにして下さい。

引数

なし

返り値

TRUE 正常終了
FALSE エラー発生中のため実行できない

参照

DWC_GetGHTTPData, DWC_GetGHTTPDataEx, DWC_GetGHTTPDataEx2, DWC_PostGHTTPData
