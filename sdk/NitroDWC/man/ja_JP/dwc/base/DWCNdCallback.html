DWCNdCallback

定義


#include <nd/dwc_nd.h>

typedef void (*DWCNdCallback)( DWCNdCallbackReason reason,
                               DWCNdError error,
                               int servererror );

説明

ダウンロードライブラリによって呼び出される通知コールバック関数を表すポインタ型です。

本関数内で、DWC_NdCleanupAsync関数を呼び出さないでください。

引数

reason 通知コールバック関数が呼び出された際どのような処理が実行中であったかを示す値
error 通知コールバック関数が呼び出された際に発生したエラーを示す値
servererror サーバから通知されたエラーコード
サーバエラーとは関係ないコールバックの場合は-1が入ります

返り値

なし

参照

DWC_NdInitAsync, DWC_NdCleanupAsync, DWCNdCallbackReason, DWCNdError
