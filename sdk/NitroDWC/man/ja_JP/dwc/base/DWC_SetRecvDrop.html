DWC_SetRecvDrop

構文


#include <dwc.h>

BOOL DWC_SetRecvDrop( u8 drop, u8 aid )

説明

DWC_SendUnreliable / DWC_SendUnreliableBitmap 関数によって送られてきたデータ受信時のパケットロス率を設定する関数です。

これはデバッグ用の関数です。 最終ROM版（FINALROM）ライブラリでは何も動作を行いません。

引数

drop パケットロス率。0〜100のパーセンテージで指定できます。
aid 送信元のAID

返り値

TRUE パケットロス率設定成功
FALSE FriendsMatchライブラリ非動作中、無効なAID指定、もしくはメモリ確保の失敗により設定失敗

参照

DWC_SetSendDrop, DWC_SendUnreliable, DWC_SendUnreliableBitmap
