DWC_NASLoginAsync

構文

#include <dwc.h>

BOOL DWC_NASLoginAsync(void);

説明

任天堂認証サーバで認証を行いWi-Fiコネクションへ接続します。本関数を呼び出した後、DWC_NASLoginProcess関数を呼び出して処理を進行させる必要があります。

本関数を呼び出すには、インターネット接続が完了している必要があります。
また、FriendsMatchライブラリを使用する場合は、DWC_LoginAsync関数で同様の処理を行ないますので、本関数を呼び出す必要はありません。

引数

なし

返り値

TRUE 認証処理を開始した。
FALSE 認証処理を開始できなかった。

参照

DWC_NASLoginProcess, FriendsMatchライブラリ, DWC_LoginAsync
