DWC_GetGsProfileId

構文

#include <dwc.h>

int DWC_GetGsProfileId( const DWCUserData* userdata,
                        const DWCFriendData* frienddata )

説明

frienddataで指定された友達情報からGSプロファイルIDを取得します。 GSプロファイルIDは、DWC_GetFriendDataType関数で取得できる友達情報のタイプが、友達登録鍵及び成立済み友達の場合に取得できます。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ
frienddata 友達情報が格納されたDWCFriendData構造体へのポインタ

返り値

正の整数 GSプロファイルID
0 友達登録鍵が不正のため、取得できない
-1 ログインIDが格納されているため、取得できない

参照

DWC_GetFriendDataType
