DWC_CheckInet

構文

#include <dwc.h>

BOOL DWC_CheckInet( void );

説明

自動接続処理の進行状態を確認します。 DWC_ProcessInet関数呼出し後に、本関数で自動接続処理が完了したかどうかを確認してください。

インターネット接続状態の詳細は、DWC_GetInetStatus関数で確認してください。

引数

なし

返り値

TRUE 自動接続処理が完了している
FALSE 自動接続処理中、もしくは自動接続開始前

参照

DWC_InitInet, DWC_SetAuthServer, DWC_ConnectInetAsync, DWC_ProcessInet, DWC_GetInetStatus, DWC_CleanupInet DWC_CleanupInetAsync
