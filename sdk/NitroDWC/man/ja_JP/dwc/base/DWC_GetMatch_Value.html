DWC_GetMatch*Value

構文

#include <dwc.h>

int DWC_GetMatchIntValue( int   index,
                          const char* keyString,
                          int   idefault );

const char* DWC_GetMatchStringValue( int   index,
                                     const char* keyString,
                                     const char* sdefault );

説明

マッチメイク指標キーに対応する値/文字列を取得します。
プレイヤー評価コールバック内での使用のみ保証しており、それ以外の場所で呼び出した場合の戻り値は不定です。

引数

index マッチメイク候補プレイヤーのインデックス。プレイヤー評価コールバックの引数 index をそのまま渡してください。
keyString キー名（キー識別のための文字列）へのポインタ
idefault 指定したキーを持っていないプレイヤーだった場合のデフォルト値
sdefault 指定したキーを持っていないプレイヤーだった場合のデフォルト文字列

返り値

マッチメイク候補プレイヤーが持つ、指定したキーに対応する値/文字列。
キーがない場合は引数 idefault/sdefault に渡した値/文字列を返します。

参照

DWC_AddMatchKey*, DWCEvalPlayerCallback
