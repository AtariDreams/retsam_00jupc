WCM_CountApList

構文


#include <nitroWiFi/wcm.h>

s32   WCM_CountApList( void );

説明

無線アクセスポイント自動探索中に発見され、内部バッファにリストアップされた機器の数を取得する同期関数です。
WCM_ClearApList 関数によって内部リストは初期化されますので、機器の数も 0 に初期化されます。 また、WCM_StartupAsync 関数呼び出し時に内部リスト用に有効なバッファを与えていない場合などには、 内部バッファへのリストアップ自体が行われませんので、この関数には必ず 0 が返されます。 WCM ライブラリを初期化する前の呼び出しにも 0 が返されます。
無線アクセスポイント自動探索中には不定期に内部リストバッファの状態は変更されていきますので、一旦 WCM_LockApList 関数によって 内部リストの更新を禁止した上で当関数を使用することをお勧めします。

引数

ありません。

返り値

内部バッファにリストアップされている無線アクセスポイント情報の数を返します。

参照

WCM_StartupAsync, WCMConfig, WCM_LockApList, WCM_ClearApList, WCM_PointApList, WCM_PointApListLinkLevel
