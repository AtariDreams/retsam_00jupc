CPS_STT_*

構文

#include <nitroWiFi/cps.h>
enum {
	CPS_STT_CLOSED = 0,
	CPS_STT_LISTEN,
	CPS_STT_SYN_SENT,
	CPS_STT_SYN_RCVD,
	CPS_STT_ESTABLISHED,
	CPS_STT_CLOSE_WAIT,
	CPS_STT_LAST_ACK,
	CPS_STT_FIN_WAIT1,
	CPS_STT_FIN_WAIT2,
	CPS_STT_CLOSING,
	CPS_STT_DATAGRAM,
	CPS_STT_PING
};

説明

CPSSoc.stateに設定される定数です。ソケットの状態を示します。ユーザはこの値を変更しないでください。

CPS_STT_CLOSED
CPS_STT_LISTEN
CPS_STT_SYN_SENT
CPS_STT_SYN_RCVD
CPS_STT_ESTABLISHED
CPS_STT_CLOSE_WAIT
CPS_STT_LAST_ACK
CPS_STT_FIN_WAIT1
CPS_STT_FIN_WAIT2
CPS_STT_CLOSING		 それぞれ対応するTCPステートを示します。
CPS_STT_DATAGRAM ソケットがUDPモードであることを示します。
CPS_STT_PING ソケットがPING(ICMP echo)モードであることを示します。
