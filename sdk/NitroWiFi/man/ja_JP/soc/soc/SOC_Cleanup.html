SOC_Cleanup

構文

#include <nitroWiFi/soc.h>
int SOC_Cleanup ( void );

引数

なし。

返り値

0 成功。
-1 全てのソケットサービスが停止していません。

注意：追加のエラーが今後のライブラリリリースで生じたり返されることがあります。すべての負の返り値は汎用のエラーとして扱ってください。

説明

すべてのオープンされているソケットをクローズし、ソケットライブラリをシャットダウンします。以降のソケット関数の呼び出しは、SOC_Startup()によってソケットライブラリが再初期化されるまでエラーを返します。

参照

SOC_Close, SOC_Startup

履歴
