SOC_ClearConfigError

構文

#include <nitroWiFi/soc.h>
#define SOC_IP_ERR_DHCP_TIMEOUT    -100
s32 SOC_ClearConfigError ( SOCInterface* interface );

引数

interface ホスト設定のエラーコードをクリアするSOCInterfaceへのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。

返り値

0 成功。

説明

ホスト設定のエラーコードをクリアします。

SOC_GetConfigError
