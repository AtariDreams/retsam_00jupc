SOC_InetAtoN

構文

#include <nitroWiFi/soc.h>
int SOC_InetAtoN ( const char* cp, SOCInAddr* inp );

引数

cp IPv4アドレスのドット10進記法。
inp ネットワーク バイトオーダーのバイナリIPv4アドレス。

返り値

cpが有効なIPv4アドレス記法であれば(inp がNULLでも)TRUEを返します。cpが有効なIPv4アドレス記法でなければFALSEを返します。

説明

IP アドレスをドット10進記法からバイナリアドレスに変換します。

参照

SOC_InetNtoA, SOC_InetPtoN, SOC_InetNtoP, SOCInAddr
