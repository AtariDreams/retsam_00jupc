SOC_GetGateway

構文

#include <nitroWiFi/soc.h>
void SOC_GetGateway ( SOCInterface* interface, u8* addr );

引数

interface デフォルト ゲートウェイアドレスを取得する SOCInterface へのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。
addr デフォルトゲートウェイアドレスを受け取るためのバッファへのポインタ。

返り値

なし。

説明

指定されたインターフェースのデフォルト ゲートウェイアドレスを取得します。
