SOC_GetHostByName

構文

#include <nitroWiFi/soc.h>
SOCHostEnt* SOC_GetHostByName( const char* name );

引数

name ホスト名もしくはドット10進表現による数値形式の文字列。

返り値

要求されたエントリーが見つかったときはSOCHostEnt構造体へのポインタを返します。さもなければNULLを返します。

説明

nameと名付けられたホストの情報を返します。

注意：SOC_GetHostByName()は再入可能でもスレッドセーフでもありません。

参照

SOC_GetHostByAddr, SOCConfig, SOCHostEnt
