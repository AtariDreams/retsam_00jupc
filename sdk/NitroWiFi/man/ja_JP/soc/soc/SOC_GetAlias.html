SOC_GetAlias

構文

#include <nitroWiFi/soc.h>
void SOC_GetAlias ( SOCInterface* interface, u8* addr );

引数

interface インターネット エイリアス アドレスを取得するSOCInterfaceへのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。
addr インターフェースのインターネット エイリアス アドレスを取得するための、バッファへのポインタ。

返り値

なし。

説明

指定されたインターフェースのインターネット エイリアス アドレスを取得します。
つまり、SOC_GetHostID() と同じ値を取得します。

参照

SOC_GetHostID, SOC_GetAddr
