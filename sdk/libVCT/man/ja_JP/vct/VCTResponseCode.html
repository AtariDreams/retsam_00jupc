VCTResponseCode

定義

#include <vct.h>

typedef enum {
	VCT_RESPONSE_OK,
	VCT_RESPONSE_BAD_REQUEST,
	VCT_RESPONSE_NOT_ACCEPTABLE,
	VCT_RESPONSE_BUSY_HERE,
	VCT_RESPONSE_TERMINATED,
	VCT_RESPONSE_DECLINE,
	VCT_RESPONSE_END
} VCTResponseCode;

定義

VCT_RESPONSE_OK 会話開始、終了要求許可
VCT_RESPONSE_BAD_REQUEST 不正なリクエストに対する応答
VCT_RESPONSE_NOT_ACCEPTABLE 接続を許可できない
VCT_RESPONSE_BUSY_HERE 相手先話中
VCT_RESPONSE_TERMINATED 会話開始キャンセル要求を許可
VCT_RESPONSE_DECLINE 会話要求を拒否

説明

SSPのレスポンスコードです。VCT_Responseで利用します。

VCT_RESPONSE_BAD_REQUEST, VCT_RESPONSE_NOT_ACCEPTABLE, VCT_RESPONSE_TERMINATEDは内部で利用します。ユーザーはこれらのコードでVCT_Responseを呼び出さないでください。
