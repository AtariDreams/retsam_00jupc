VCTMode

定義

#include <vct.h>

typedef enum {
    VCT_MODE_NULL,
    VCT_MODE_PHONE,
    VCT_MODE_TRANSCEIVER,
    VCT_MODE_CONFERENCE
} VCTMode;

定義

VCT_MODE_NULL モード未設定
VCT_MODE_PHONE 電話モード
VCT_MODE_TRANSCEIVER トランシーバーモード
VCT_MODE_CONFERENCE カンファレンスモード

説明

VoiceChatライブラリの現在のモードを示す列挙型です。

参照

VCT_Init, VCTConfig
