VCTAudioInfo

定義

#include <vct.h>

typedef struct VCTAudioInfo
{
	u32         internalLatency;    
	u32         bufferLatency;      
	u32         bufferCount;        
	int         clockSkew;          
	u32         sequence;           
	u32         dropCount;          
	u32         jitter;             
	u32         jamCount;           
	u32         recoverCount;       
	VCTCodec    codec;              
} VCTAudioInfo;

定義

internalLatency 受信内部遅延量（ticks値）
bufferLatency 現在のジッタバッファの遅延量（ticks値）
bufferCount 現在音声バッファに格納されているサンプル数
clockSkew clock-skewモニタの値（ticks値）
sequence 現在のオーディオシーケンス
dropCount ドロップしたパケットの数
jitter 最大ジッタ値（ticks値）
jamCount パケットが順序どおりに到達しなかった回数
recoverCount ドロップしたパケットを隠蔽処理した回数
codec 送信元のオーディオフォーマット

説明

VCT_GetAudioInfoで取得できる音声ストリーミング情報を保存する構造体。
