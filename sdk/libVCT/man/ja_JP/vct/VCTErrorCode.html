エラーコード一覧

定義

#include <vct.h>

#define VCT_ERROR_NONE                  0
#define VCT_ERROR_BAD_PARAM            -1
#define VCT_ERROR_BAD_MODE             -2
#define VCT_ERROR_BAD_REQUEST          -3
#define VCT_ERROR_SEND_FAIL            -4
#define VCT_ERROR_TRANSCEIVER_BUSY     -5
#define VCT_ERROR_EXHAUST_CLIENTS      -6

定義

VCT_ERROR_NONE エラーなし
VCT_ERROR_BAD_PARAM 不正な関数パラメータ
VCT_ERROR_BAD_MODE 不正なモード（電話モードでトランシーバー関数を呼び出した、など）
VCT_ERROR_BAD_REQUEST 不正なリクエスト
VCT_ERROR_SEND_FAIL 送信処理に失敗
VCT_ERROR_TRANSCEIVER_BUSY トランシーバーが現在BUSY（他のクライアントが会話中）
VCT_ERROR_EXHAUST_CLIENTS クライアント数が上限に達した

説明

VoiceChatライブラリのエラーコードです。
