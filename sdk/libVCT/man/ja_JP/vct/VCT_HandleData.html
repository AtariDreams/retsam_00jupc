VCT_HandleData

構文

#include <vct.h>
BOOL VCT_HandleData ( u8 aid, u8 *buffer, int size );

引数

aid 送信元aid
buffer 受信したデータ
size 受信データサイズ

返り値

TRUE 受信したデータがVoiecChatのもの
FALSE 受信したデータがVoiecChatのものではない、またはVoiceChatライブラリが初期化されていない

説明

VoiceChatのデータを処理します。DWC_SetUserRecvCallbackで指定する受信コールバックの中で呼び出してください。

bufferにはDWC_SetUserRecvCallbackから渡されたデータを渡し、sizeにバイト数を指定します。

戻り値がTRUEの場合（受信したデータがVoiceChatデータの場合）、この関数から指定したコールバック関数が呼び出されますので、追加で処理を行う必要はありません。戻り値がFALSEの場合はVoiceChat以外のデータ（ゲーム上でやりとりするデータ）ですので、ゲームで必要な処理を実行するように実装してください。

参照

VCT_Cleanup, VCTConfig
