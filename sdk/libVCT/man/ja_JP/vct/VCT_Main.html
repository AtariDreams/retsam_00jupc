VCT_Main

構文

#include <vct.h>
void VCT_Main( void );

引数

なし

返り値

なし

説明

VoiceChatのメインループを実行します。VCT_SendAudioによってキューイングされている送信音声のVAD、エコーキャンセル、エンコード、および送信処理、VCT_HandleDataで受信したデータのデコード処理、およびタイムアウトのチェックを行っています。

定期的な送信が行われるよう、この関数を必ず１ピクチャーフレームに１回呼び出してください。

割り込み処理ルーチンからは、本関数を呼び出さないでください。
