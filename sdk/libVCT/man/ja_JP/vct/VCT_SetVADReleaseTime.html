VCT_SetVADReleaseTime

構文

#include <vct.h>
void VCT_SetVADReleaseTime( int count );

引数

count リリース時間（パケット数）

返り値

なし

説明

VAD(Voice Activity Detection)が無音と判定してから、実際にパケットの送出を停止するまでの期間を設定します。

会話途中の息継ぎなどでVADが無音を判定してしまうことを避けるために、500ms以上のリリース時間を設定してください。

countに無音を判定してから実際に送信をやめるまでの時間をパケット単位で指定します。10を設定すると68 * 10 = 680ms となります。なお、デフォルトは15パケット分（68 * 15 = 1020ms）となっております。
