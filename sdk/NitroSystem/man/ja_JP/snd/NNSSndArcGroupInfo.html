NNSSndArcGroupInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcGroupInfo
{
    u32 count;
    struct NNSSndArcGroupItem item[ 0 ];
} NNSSndArcGroupInfo;






      
要素

      
      







count
グループ項目の個数です。




item
グループ項目の配列です。







    



      
説明

      

サウンドアーカイブ中のグループ情報構造体です。 


      

NNS_SndArcGetGroupInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


      

NNS_SndArcLoadGroup関数では、
この情報に基づいて、サウンドデータをロードします。


    


参照


NNSSndArcGroupItem, NNS_SndArcGetGroupInfo, NNS_SndArcLoadGroup
