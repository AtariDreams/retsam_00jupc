NNSSndArcSeqInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcSeqInfo
{
    u32 fileId;
    struct NNSSndSeqParam param;
} NNSSndArcSeqInfo;






      
要素

      
      







fileId
サウンドアーカイブ中のファイルIDです。




param
シーケンスパラメータ構造体です。







    



      
説明

      

サウンドアーカイブ中のシーケンス情報構造体です。 


      

NNS_SndArcGetSeqInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


    


参照


NNSSndSeqParam, NNS_SndArcGetSeqInfo
