NNS_SndAllocAlarm

構文

#include <nnsys/snd.h>
int NNS_SndAllocAlarm( void );

引数

なし。

返り値

サウンドアラームの取得に成功すると、アラーム番号を返します。失敗すると-1を返します。

説明

サウンドアラームを取得します。取得したサウンドアラームに対しては、 ライブラリ内部で処理を行いません。

取得したサウンドアラームを解放するためには、 NNS_SndFreeAlarm関数を呼びだします。

注意

サウンドアラームは、チャンネルやキャプチャと同期して処理を行うために使用する コールバックオブジェクトです。 詳しくはサウンドドライバのマニュアルをご覧くさい。
