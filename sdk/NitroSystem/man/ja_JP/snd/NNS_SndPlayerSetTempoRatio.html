NNS_SndPlayerSetTempoRatio

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetTempoRatio( NNSSndHandle* handle, int ratio );

引数

handle サウンドハンドルのポインタです。
ratio テンポの倍率です。

返り値

なし。

説明

シーケンスのテンポを変更します。

サウンドハンドルhandleに結びつけられたシーケンスのテンポを変更します。 サウンドハンドルが無効の場合は、何もしません。

元々のテンポを、ratio/256倍します。 （呼ぶたびに、累積されるわけではありません）

参照

なし。
