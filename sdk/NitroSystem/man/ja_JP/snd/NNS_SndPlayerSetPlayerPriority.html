NNS_SndPlayerSetPlayerPriority

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetPlayerPriority(
        NNSSndHandle* handle,
        int priority );

引数

handle サウンドハンドルのポインタです。
priority プレイヤープライオリティの値です。値の範囲は 0〜127で、大きいほど優先度が高くなります。

返り値

なし。

説明

シーケンスのプレイヤープライオリティを変更します。

サウンドハンドルhandleに結びつけられたシーケンスのプレイヤープライオリティを変更します。 サウンドハンドルが無効の場合は、何もしません。

シーケンス再生時にプレイヤープライオリティを指定するときは、 NNS_SndArcPlayerStartSeqEx関数または、 NNS_SndArcPlayerStartSeqArcEx関数 を使います。 この関数は、再生後にプレイヤープライオリティを変更したい場合に、使用します。

参照

NNS_SndArcPlayerStartSeqEx, NNS_SndArcPlayerStartSeqArcEx
