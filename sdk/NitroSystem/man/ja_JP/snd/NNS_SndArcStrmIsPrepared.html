NNS_SndArcStrmIsPrepared

構文

#include <nnsys/snd.h>
BOOL NNS_SndArcStrmIsPrepared( NNSSndStrmHandle* handle );

引数

handle ストリームハンドルのポインタです。

返り値

準備が完了したかどうかを返します。

説明

ストリームの準備が完了したかどうかを調べます。 ストリームハンドルhandleが無効の場合は、FALSEを返します。

あらかじめ、NNS_SndArcStrmPrepare関数で 準備を行ったストリームが、即座に再生を開始できる状態かどうかを調べます。
