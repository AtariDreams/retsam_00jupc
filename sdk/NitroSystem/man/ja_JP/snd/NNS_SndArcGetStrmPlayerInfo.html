NNS_SndArcGetStrmPlayerInfo

構文

#include <nnsys/snd.h>
const NNSSndArcStrmPlayerInfo* NNS_SndArcGetStrmPlayerInfo(
        int playerNo );

引数

playerNo プレイヤー番号です。

返り値

ストリームプレイヤー情報構造体NNSSndArcStrmPlayerInfoの ポインタを返します。 無効なプレイヤー番号を指定した場合は、NULLを返します。

説明

ストリームプレイヤー情報構造体を取得します。
