NNS_SndWaveOutSetSpeed

構文

#include <nnsys/snd.h>
void NNS_SndWaveOutSetSpeed(
        NNSSndWaveOutHandle handle,
        int speed );

引数

handle 波形再生ハンドルです。
speed 再生スピードです。

返り値

なし。

説明

波形再生のスピードを変更します。 すでに停止中の波形再生ハンドルhandleに対し、この関数を呼びだしても何もしません。

再生スピードspeedは、再生する速さを指定します。 speed÷32768倍の速さで再生されます。 再生スピードを速くすると、再生時間が短くなり、再生される音の高さが高くなります。

この関数は、再生中に再生スピードを変更するために呼びだします。 再生開始時の再生スピードを設定したい場合は、 NNS_SndWaveOutStart関数 の引数で設定してください。
