NNSSndArcStrmInfo

定義

#include <nnsys/snd.h>
typedef struct NNSSndArcStrmInfo
{
    u32 fileId;
    u8 volume;
    u8 playerPrio;
    u8 playerNo;
    u8 flags;
 } NNSSndArcStrmInfo;






      
要素

      
      







fileId
サウンドアーカイブ中のファイルIDです。




volume
初期ボリューム値です。




playerPrio
プレイヤープライオリティです。




playerNo
プレイヤー番号です。




flags
オプションフラグです。







    



      
説明

      

サウンドアーカイブ中のストリーム情報構造体です。 


      

NNS_SndArcGetStrmInfo関数を使って、
サウンドアーカイブからこの構造体を取得することができます。


      

オプションフラグflagsは、以下の値の論理和です。


      
      
表 . オプションフラグ







ラベル
値
説明






NNS_SND_ARC_STRM_FORCE_STEREO
1<<0
モノラルストリームデータをステレオプレイヤーで再生したときに、
２チャンネル使って、より大きな音量で再生します。
