NNS_SndStrmHandleIsValid

構文

#include <nnsys/snd.h>
BOOL NNS_SndStrmHandleIsValid( const NNSSndStrmHandle* handle );

引数

handle ストリームハンドルのポインタです。

返り値

ストリームハンドルにストリームが結びつけられているかどうかを返します。

説明

ストリームハンドルが有効かどうか調べます。ストリームハンドルが有効とは、ストリームが結びつけられた状態を指します。

参照

なし。
