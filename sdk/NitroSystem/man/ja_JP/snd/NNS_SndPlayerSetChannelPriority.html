NNS_SndPlayerSetChannelPriority

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetChannelPriority(
        NNSSndHandle* handle,
        int priority );

引数

handle サウンドハンドルのポインタです。
priority 発音プライオリティの値です。値の範囲は 0〜127で、大きいほど優先度が高くなります。

返り値

なし。

説明

シーケンスの発音プライオリティを変更します。

サウンドハンドルhandleに結びつけられたシーケンスの発音プライオリティを変更します。 サウンドハンドルが無効の場合は、何もしません。

注意

この関数は、NNS_SndArcPlayerStartSeq*関数と NNS_SndArcPlayerStartSeqArc*関数内で 呼びだされています。 再度、この関数を呼びだすと、設定した値が上書きされます。

参照

NNS_SndArcPlayerStartSeq*, NNS_SndArcPlayerStartSeqArc*

履歴
