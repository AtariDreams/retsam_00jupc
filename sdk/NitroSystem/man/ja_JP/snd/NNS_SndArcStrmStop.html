NNS_SndArcStrmStop

構文

#include <nnsys/snd.h>
void NNS_SndArcStrmStop( NNSSndStrmHandle* handle, int fadeFrame );

引数

handle ストリームハンドルのポインタです。
fadeFrame フェードアウトフレームです。値の範囲は、0 〜 65535です。

返り値

なし。

説明

ストリームハンドルhandleに、結びつけられたストリームを停止します。 ストリームハンドルhandleが無効の場合は、何もしません。

fadeFrameで指定したゲームフレーム分の時間をかけて、徐々に音が小さくなった後、 ストリームは完全に停止します。fadeFrameが0の場合は、即座に停止します。 フェードアウト中のプライオリティは、最低の0になります。

参照

なし。
