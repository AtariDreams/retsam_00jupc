NNS_SndPlayerSetPlayerVolume

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetPlayerVolume( int playerNo, int volume );

引数

playerNo プレイヤー番号です。
volume ボリュームの値です。値の範囲は、0〜127です。

返り値

なし。

説明

指定プレイヤーのボリュームを設定します。 デフォルトでは、最大の127が設定されています。

この関数で設定されたボリュームは、 指定プレイヤーで再生中のシーケンス及び、指定プレイヤーでこれから再生される全てのシーケンスに対して効果があります。 また、他のいかなるボリュームパラメータとも独立して効果を発揮し、それらは、累積して影響します。

参照

なし。
