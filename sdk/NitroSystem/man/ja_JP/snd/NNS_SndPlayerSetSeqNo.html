NNS_SndPlayerSetSeqNo

構文

#include <nnsys/snd.h>
void NNS_SndPlayerSetSeqNo( NNSSndHandle* handle, int seqNo );

引数

handle サウンドハンドルのポインタです。
seqNo シーケンス番号です。

返り値

なし。

説明

シーケンス番号を設定します。

サウンドハンドルhandleに、結びつけられたシーケンスのシーケンス番号を設定します。サウンドハンドルが無効の場合は、何もしません。

シーケンス番号を設定すると、後で、設定したシーケンス番号を指定して、シーケンスを止めたりすることができるようになります。 詳しくは、NNS_SndPlayerStopSeqBySeqNo関数や、 NNS_SndPlayerCountPlayingSeqBySeqNo関数を参照してください。

注意

この関数は、NNS_SndArcPlayerStartSeq*関数内で、 呼びだされています。再度、この関数を呼びだすと、設定した値が上書きされます。

この関数と、NNS_SndPlayerSetSeqArcNo関数は、 どちらか一方のみ呼びだします。両方呼びだすと、後から設定した方が有効になります。

参照

NNS_SndPlayerStopSeqBySeqNo, NNS_SndPlayerCountPlayingSeqBySeqNo, NNS_SndArcPlayerStartSeq*, NNS_SndPlayerSetSeqArcNo
