NNS_FndInitList

構文

#include <nnsys/fnd.h>
void NNS_FndInitList( NNSFndList* list, u16 offset );

引数

list リスト構造体へのポインタ
offset NNSFndLink型メンバ変数のオブジェクト内でのオフセット値

返り値

なし。

説明

リスト構造体を初期化し、オブジェクトを繋ぐことができる状態にします。 offsetに指定する値は、stddef.hで定義されるoffsetofマクロで求める事が出来ます。 この値は、リストに接続されているオブジェクト（構造体）へのポインタから、 そのオブジェクトがメンバ変数として持っているNNSFndLinkへのポインタに変換するために使用されます。
