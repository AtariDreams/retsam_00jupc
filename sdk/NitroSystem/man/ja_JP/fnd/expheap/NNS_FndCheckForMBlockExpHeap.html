NNS_FndCheckForMBlockExpHeap

構文

#include <nnsys/fnd.h>
BOOL  NNS_FndCheckForMBlockExpHeap(
          const void*         memBlock,
          NNSFndHeapHandle    heap,
          u32                 optFlag);

引数

memBlock チェックするメモリブロックへのポインタ
heap 拡張ヒープのハンドル。NULLを指定することも出来ます。NULLを指定した場合は、メモリブロックのアドレスの範囲のチェックやサイズのチェックはおおまかに行います。
optFlag チェックについてのオプション(詳細は下記を参照)

返り値

拡張ヒープのメモリブロックにエラーが発見されなかった場合、TRUE を返します。エラーが発見された場合、FALSEを 返します。

説明

拡張ヒープのメモリブロックが破壊されていないかどうかのチェックを行います。具体的には、拡張ヒープのメモリブロックが持つ管理領域が正常であるかをチェックします。

optFlag で、チェックのオプションを指定できます。

NNS_FND_HEAP_ERROR_PRINT メモリブロックにエラーを発見したとき、エラーの内容をOS_Printf()関数で出力します。

この関数はデバッグ用です。最終ROM版(FINALROM)ライブラリでは何も行わず、常に TRUE を返します。
