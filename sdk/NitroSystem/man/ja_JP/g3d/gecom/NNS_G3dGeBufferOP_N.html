NNS_G3dGeBufferOP_N

構文

#include <nnsys/g3d/gecom.h>
void
NNS_G3dGeBufferOP_N(
    u32 op,
    const u32* args,
    u32 num
);

引数

op ジオメトリコマンド
args 引数リストへのポインタ
num 引数の数

返り値

なし。

説明

num個の引数を持つジオメトリコマンドを安全にジオメトリエンジンに書き出します。NNS_G3dGeSendDL関数でディスプレイリストが転送中の場合、ジオメトリコマンドバッファにコマンドをバッファリングして復帰します。コマンドバッファが設定されていないか、コマンドバッファの容量が足りない場合はウェイト処理が入ります。

参照

NNS_G3dGeSendDL

履歴
