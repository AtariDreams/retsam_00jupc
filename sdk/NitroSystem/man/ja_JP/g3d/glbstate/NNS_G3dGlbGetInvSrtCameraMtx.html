NNS_G3dGlbGetInvWV

構文

#include <nnsys/g3d/glbstate.h>
const MtxFx43* NNS_G3dGlbGetInvSrtCameraMtx( void );
const MtxFx43* NNS_G3dGlbGetInvWV(void);

引数

なし。

返り値

カメラ行列にBaseSRT行列をかけた行列の逆行列へのポインタを返します。

説明

G3Dライブラリ内部で保持しているグローバル状態に設定されているカメラ行列に BaseSRT行列をかけた行列の逆行列へのポインタを返します。 BaseSRT行列をかけた行列が計算されていなかった場合には、 その行列と逆行列を求めてから逆行列へのポインタを返します。
