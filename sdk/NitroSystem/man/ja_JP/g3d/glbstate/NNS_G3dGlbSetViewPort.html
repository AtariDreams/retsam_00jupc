NNS_G3dGlbSetViewPort

構文

#include <nnsys/g3d/glbstate.h>
void
NNS_G3dGlbSetViewPort(
    int x1,
    int y1,
    int x2,
    int y2
);

引数

x1 左下のＸ座標
y1 左下のＹ座標
x2 右上のＸ座標
y2 右上のＹ座標

返り値

なし。

説明

G3Dライブラリ内部で保持しているグローバル状態にビューポートを設定します。 ビューポートの設定座標値は、X座標は0〜255、Y座標は0〜191です。

参照

NNS_G3dGlbGetViewPort
