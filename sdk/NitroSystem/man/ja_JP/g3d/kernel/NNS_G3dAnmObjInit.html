NNS_G3dAnmObjInit

構文

#include <nnsys/g3d/kernel.h>
void
NNS_G3dAnmObjInit(
    NNSG3dAnmObj*       pAnmObj,
    void*               pResAnm,
    const NNSG3dResMdl* pResMdl,
    const NNSG3dResTex* pResTex
);

引数

pAnmObj AnmObj構造体へのポインタ
pResAnm アニメーションリソースへのポインタ
pResMdl モデルリソースへのポインタ
pResTex テクスチャリソースへのポインタ

返り値

なし。

説明

NNSG3dAnmObj構造体を初期化します。 pAnmObjで指定されるアニメーションオブジェクトは、最低でNNS_G3dAnmObjCalcSizeRequired 関数で得られるサイズ分の大きさを持っている必要があります。 pResTexは、テクスチャパターンアニメーション用のアニメーションオブジェクトを初期化する場合以外ではNULLでも構いません。

参照

NNS_G3dAnmObjCalcSizeRequired
