NNS_G3dBindMdlPlttEx

構文

#include <nnsys/g3d/kernel.h>
BOOL
NNS_G3dBindMdlPlttEx(
    NNSG3dResMdl* pMdl,
    const NNSG3dResTex* pTex,
    const NNSG3dResName* pResName
);

引数

pMdl モデルリソースへのポインタ
pTex テクスチャリソースへのポインタ
pResName テクスチャ名へのポインタ

返り値

バインドが成功した場合TRUEを返します。

説明

モデル内で、pResNameというパレット名に関連付けられているマテリアルのパレットエントリを テクスチャブロック内の名前pResNameを持つパレットに関連付けます。

参照

NNS_G3dBindMdlPltt, NNS_G3dForceBindMdlPltt
NNS_G3dReleaseMdlPltt, NNS_G3dReleaseMdlPlttEx
