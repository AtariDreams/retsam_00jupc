NNS_G3dRenderObjSetBlendFuncVis

構文

#include <nnsys/g3d/kernel.h>

void
NNS_G3dRenderObjSetBlendFuncVis(
    NNSG3dRenderObj* pRenderObj,
    NNSG3dFuncAnmBlendVis func
);

引数

pRenderObj NNSG3dRenderObj構造体へのポインタ
func ビジビリティ計算結果のブレンド関数へのポインタ

返り値

なし。

説明

レンダリングオブジェクトにビジビリティ計算結果のブレンド関数を設定します。ここで設定されたブレンド関数は、NNS_G3dDraw関数による描画時にＧ３Ｄから呼び出されます。レンダリングオブジェクトの初期化時にデフォルトのブレンド関数がレンダリングオブジェクトに設定されていますので、デフォルトのブレンド関数を使用する場合はこの関数を呼び出す必要はありません。
