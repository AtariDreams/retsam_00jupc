NNS_G3dReleaseMdlTexEx

構文

#include <nnsys/g3d/kernel.h>
BOOL
NNS_G3dReleaseMdlTexEx(
    NNSG3dResMdl*        pMdl,
    const NNSG3dResName* pResName
);

引数

pMdl モデルリソースへのポインタ
pResName テクスチャ名へのポインタ

返り値

テクスチャのリリースに成功すればTRUEを返します。

説明

pMdlで指定されたモデル内で、名前pResName という名前をもつテクスチャに関連付けられているマテリアルのテクスチャエントリの関連付けを解除します。

参照

NNS_G3dBindMdlTex, NNS_G3dBindMdlTexEx, NNS_G3dForceBindMdlTex
NNS_G3dReleaseMdlTex
