NNS_G3dGetMdlByName

構文

#include <nnsys/g3d/binres/res_struct_accessor.h>
NNSG3dResMdl*
NNS_G3dGetMdlByName(
    const NNSG3dResMdlSet* mdlSet,
    const NNSG3dResName*   name
);

引数

mdlSet モデルセットへのポインタ
name リソース名へのポインタ

返り値

モデルへのポインタを返します。

説明

モデルセットmdlSet内のリソース名nameを持つモデルを取得します。
