NNS_G3dSetResultMtx

構文

#include <nnsys/g3d/util.h>
BOOL
NNS_G3dSetResultMtx(
    const NNSG3dRenderObj* pRenderObj,
    const MtxFx43*         pos,
    const MtxFx33*         nrm,
    u32                    nodeID
);

引数

pRenderObj RenderObj構造体へのポインタ
pos クリップ座標行列へのポインタ
nrm 方向ベクトル行列へのポインタ
nodeID ノードID

返り値

行列がスタックの該当位置に代入出来た場合はTRUEを返します。

説明

NNS_G3dDraw関数の実行後にnodeIDに対応する行列が行列スタックに残るような設定になっている場合は（g3dcvtr で-sオプションをつけた場合は全てのnodeIDに対応する行列が行列スタックにストアされている）、 pos, nrmで指定された行列をスタックの該当位置に代入します。 代入できた場合はTRUE,できなかった場合はFALSEが返ります。

戻り値がTRUEの場合、行列モードはPosition/Vector行列モードに変更されています。 nrmはNULLでもかまいませんが、posはNULLであってはいけません。
