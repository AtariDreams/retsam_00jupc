NNS_G3dSearchSbcCmd

構文

#include <nnsys/g3d/util.h>
const u8*
NNS_G3dSearchSbcCmd(
    const u8* c,
    u8        cmd
);

引数

c SBCコード列へのポインタ
cmd 探すSBC

返り値

見つかったSBCを指す、SBCコード列内のポインタを返します。

説明

SBCコマンド(cmd)を、cで指定されたSBCコード列の中から見つけます。 見つかった場合はSBCコード列内のポインタを返します。 見つからなかった場合はNULLを返します。 SBCコマンド内のフラグフィールド(上位3ビット)は無視されます。
