NNS_McsInitFileIO

構文

#include <nnsys/mcs.h>
void NNS_McsInitFileIO(void* workMem);

引数

workMem MCS File I/O が使用するワーク用メモリ。バイト数 が NNS_MCS_FILEIO_WORKMEM_SIZE で、アライメントが 4 のメモリ領域を指定してください。

返り値

なし。

説明

PC上のファイルを読み書きするAPIとディレクトリ内のファイルを列挙するAPIのための初期化を行います。

この関数Iを呼び出す前に、NNS_McsInit()を呼び出してmcsライブラリを初期化しておく必要があります。
