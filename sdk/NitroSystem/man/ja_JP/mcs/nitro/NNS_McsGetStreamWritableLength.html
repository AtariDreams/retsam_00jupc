NNS_McsGetStreamWritableLength

構文

#include <nnsys/mcs.h>
BOOL NNS_McsGetStreamWritableLength(u32* pLength);

引数

pLength 一度に送信可能なサイズ(バイト数)を入れる変数へのポインタ。

返り値

関数が成功した場合は TRUE、失敗した場合は FALSE が返ります。

説明

一度に送信可能なサイズ(バイト数)を取得します。関数NNS_McsWriteStreamで一度に送信可能なサイズを超えたデータを書き込む場合、PC側で書き込んだデータを読み取るまで関数呼び出しはブロックされ、制御を返しません。

参照

NNS_McsWriteStream
