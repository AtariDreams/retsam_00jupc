NNS_G2dFindBinaryBlock

構文

#include <nnsys/g2d/g2d_load.h>

    NNSG2dBinaryBlockHeader* NNS_G2dFindBinaryBlock
    ( 
        NNSG2dBinaryFileHeader* pBinFileHeader, 
        u32                     signature 
    );

引数

pBinFileHeader [IN] バイナリファイルヘッダ
signature [IN] バイナリブロック識別子

返り値

指定バイナリブロック識別子と一致した最初のブロック

説明

バイナリファイルから指定した ブロック識別子を持つバイナリブロックを取得します。
取得できなかった場合はNULLを返します。

参照

なし
