NNS_G2dGetRendererOverwrite*

構文

#include <nnsys/g2d/g2d_Renderer.h>

    
    NNS_G2D_INLINE u16 NNS_G2dGetRendererOverwritePriority
    (
        const NNSG2dRendererInstance* pRend
    );

    NNS_G2D_INLINE u16 NNS_G2dGetRendererOverwritePlttNo
    (
        const NNSG2dRendererInstance* pRend    
    );

    NNS_G2D_INLINE BOOL NNS_G2dGetRendererOverwriteMosaicFlag
    (
        const NNSG2dRendererInstance* pRend    
    );
    
    NNS_G2D_INLINE GXOamMode NNS_G2dGetRendererOverwriteOBJMode
    (
        const NNSG2dRendererInstance* pRend    
    );

引数

pRend レンダラ 実体

返り値

レンダラがOAMの上書きに使用する各種パラメータ。

説明

レンダラが描画するOBJに上書きするパラメータの値を取得します。
NNS_G2dGetRendererOverwritePriority()は描画優先度を取得します。
NNS_G2dGetRendererOverwritePlttNo()はパレット番号を取得します。
NNS_G2dSetRendererOverwriteMosaicFlag()はモザイクフラグを取得します。
NNS_G2dSetRendererOverwriteOBJMode()はオブジェモードを取得します。

パラメータ上書き機能を有効にするために、NNS_G2dSetRendererOverwriteEnable()で適切なパラメータの上書き機能を有効に設定する必要があります。

参照

NNS_G2dSetRendererOverwriteEnable/Disable
 NNS_G2dSetRendererOverwrite*
