NNS_G2dGetUserExAnimSequenceAttr

構文

#include <nnsys/g2d/fmt/g2d_Anim_data.h>

static inline const NNSG2dUserExAnimSequenceAttr* 
NNS_G2dGetUserExAnimSequenceAttr
( 
    const NNSG2dUserExAnimAttrBank* pAnimAttrBank,
    u16                             idx 
);

引数

pAnimAttrBank [IN] アニメーションユーザ拡張アトリビュートバンク
idx [IN] アニメーションシーケンスの番号

返り値

アニメーションシーケンスユーザ拡張アトリビュート

説明

ユーザ拡張アトリビュートバンク から シーケンスユーザ拡張アトリビュート を取得します。
不正な番号が指定された場合は、NULLを返します。
ユーザ拡張アトリビュートを含むデータを出力するためには、-oua オプションを指定してデータを出力する必要があります。
