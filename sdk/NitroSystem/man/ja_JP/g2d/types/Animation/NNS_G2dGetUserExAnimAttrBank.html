NNS_G2dGetUserExAnimAttrBank

構文

#include <nnsys/g2d/fmt/g2d_Anim_data.h>

static inline const NNSG2dUserExAnimAttrBank* 
NNS_G2dGetUserExAnimAttrBank( const NNSG2dAnimBankData* pAnimBank )

引数

pAnimBank [IN] アニメーション情報バンク

返り値

アニメーションユーザ拡張アトリビュートバンク

説明

アニメーション情報バンク から アニメーションユーザ拡張アトリビュートバンク を取得します。
取得に失敗した場合は、NULLが返ります。
ユーザ拡張アトリビュートを含むデータを出力するためには、-oua オプションを指定してデータを出力する必要があります。

参照

なし
