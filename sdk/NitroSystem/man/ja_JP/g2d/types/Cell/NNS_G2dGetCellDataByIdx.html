NNS_G2dGetCellDataByIdx

構文

#include <nnsys/g2d/load/g2d_NCE_load.h>

const NNSG2dCellData* NNS_G2dGetCellDataByIdx( 
    const NNSG2dCellDataBank*   pCellData, 
    u16                         idx );

引数

pCellData [IN] セルデータバンク
idx [IN] インデックス

返り値

セルデータ

説明

セルデータバンクからインデックス指定で、セルデータを取得します。
不正なインデックスが指定された場合はNULLを返します。

参照

なし
