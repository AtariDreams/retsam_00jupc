NNS_G2dGetUserExCellAttrValue

構文

#include <nnsys/g2d/fmt/g2d_Cell_data.h>

static inline const u32
NNS_G2dGetUserExCellAttrValue
( 
    const NNSG2dUserExCellAttr* pCellAttr 
);

引数

pCellAttr [IN] セルユーザ拡張アトリビュート

返り値

アトリビュート値

説明

セルユーザ拡張アトリビュートから実際のアトリビュート値を取得します。
ユーザ拡張アトリビュートを利用するためには、-oua オプションを指定してデータをコンバートする必要があります。
