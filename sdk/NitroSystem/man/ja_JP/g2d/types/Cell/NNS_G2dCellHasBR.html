NNS_G2dCellHasBR

構文

#include <nnsys/g2d/fmt/g2d_Cell_data.h>

    static inline BOOL NNS_G2dCellHasBR( const NNSG2dCellData* pCell );

引数

pCell [IN] セルデータ

返り値

セルが境界矩形(s16フォーマット)情報をもつかどうか。持っていればTRUE。

説明

セルが境界矩形を持つかどうかを取得します。
境界矩形情報を持っていればTRUEを返します。
本関数がTRUEを返した場合は以下のキャストが有効です。


    NNSG2dCellDataWithBR* pBR = ( NNSG2dCellDataWithBR* )pCell;

境界矩形情報を持つセルデータを作成するには、-br オプションを指定してコンバートを行ってください。

参照

NNS_G2dGetCellBoundingRect
