NNS_G2dGetUserExCellAttr

構文

#include <nnsys/g2d/fmt/g2d_Cell_data.h>

static inline const NNSG2dUserExCellAttr* 
NNS_G2dGetUserExCellAttr
( 
    const NNSG2dUserExCellAttrBank* pCellAttrBank, 
    u16                             idx
);

引数

pCellAttrBank [IN] セルユーザ拡張アトリビュートバンク
idx [IN] セル番号

返り値

セルユーザ拡張アトリビュート

説明

セル拡張アトリビュートバンク中のセルユーザ拡張アトリビュートをセル番号を指定して取得します。
取得できなかった場合はNULLを返します。
ユーザ拡張アトリビュートを利用するためには、-oua オプションを指定してデータをコンバートする必要があります。

なし
