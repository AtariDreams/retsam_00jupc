NNS_G2dSetCurrentPolygonID

構文

#include <nnsys/g2d/g2d_Softsprite.h>
void NNS_G2dSetCurrentPolygonID( u16 id );

引数

id [IN] ソフトウェアスプライト描画時に使用するポリゴンID

返り値

なし

説明

ソフトウェアスプライト描画時に使用するポリゴンIDを設定します
本関数は NNS_G2dSetSpriteCurrentPolygonID() に 改名されました。本関数は、過去のバージョンとの互換性維持のため、別名としてのこされます。

NNS_G2dSetSpriteCurrentPolygonID
