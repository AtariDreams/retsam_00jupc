NNS_G2dTextCanvasSetVSpace

構文


#include <nnsys/g2d/g2d_TextCanvas.h>

void NNS_G2dTextCanvasSetVSpace(
	NNSG2dTextCanvas* pTxn,
	int vSpace
);

引数

pTxn [IN] TextCanvas へのポインタ
vSpace [IN] 新しく設定する行間隔 (ピクセル単位)

返り値

なし。

説明

TextCanvas が用いる行間隔を変更します。

参照

NNS_G2dTextCanvasGetVSpace
