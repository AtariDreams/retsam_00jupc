NNS_G2dFontFindGlyphIndex

構文


#include <nnsys/g2d/g2d_Font.h>

u16 NNS_G2dFontFindGlyphIndex(
	const NNSG2dFont* pFont,
	u16 c
);

引数

pFont [IN] フォント へのポインタ
c [IN] グリフインデックスを取得する文字の文字コード

返り値

c に対応するグリフが存在する場合は、そのグリフインデックスを返します。
対応するグリフが存在しない場合は NNS_G2D_GLYPH_INDEX_NOT_FOUND を返します。

説明

c に対応するグリフのグリフインデックスを探します。

NNS_G2dFontGetGlyphIndex
