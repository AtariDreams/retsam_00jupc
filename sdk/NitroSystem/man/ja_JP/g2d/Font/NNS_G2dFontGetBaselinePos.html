NNS_G2dFontGetBaselinePos

構文


#include <nnsys/g2d/g2d_Font.h>

int NNS_G2dFontGetBaselinePos(
	const NNSG2dFont* pFont
);

引数

pFont [IN] フォント へのポインタ

返り値

セルの上端からベースラインまでのピクセル数を返します。

説明

セルの上端からベースラインまでのピクセル数を取得します。

ベースラインに太さはなくピクセルとピクセルの間に存在します。 セル中のベースラインの位置はフォント中の全てのグリフで同じです。

縦書きフォントではベースラインはグリフの左部に縦に存在します。返り値は文字の右端からベースラインまでのピクセル数になります。
縦持ちフォントでは返り値は文字の上端からベースラインまでのピクセル数になります。
