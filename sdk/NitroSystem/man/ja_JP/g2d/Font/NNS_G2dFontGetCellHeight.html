NNS_G2dFontGetCellHeight

構文


#include <nnsys/g2d/g2d_Font.h>

u8 NNS_G2dFontGetCellHeight(
	const NNSG2dFont* pFont
);

引数

pFont [IN] フォント へのポインタ

返り値

フォントのセルの高さを返します。(ピクセル単位)

説明

フォントのセルの高さを取得します。

NITROフォントでは個々のグリフイメージがセルと呼ばれるビットマップに格納されています。 セルはフォント内の全てのグリフで同じ大きさです。

参照

NNS_G2dFontGetHeight, NNS_G2dFontGetLineFeed
