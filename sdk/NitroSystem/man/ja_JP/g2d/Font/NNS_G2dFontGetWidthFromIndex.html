NNS_G2dFontGetWidthFromIndex

構文


#include <nnsys/g2d/g2d_Font.h>

int NNS_G2dFontGetWidthFromIndex(
	const NNSG2dFont* pFont,
	u16 idx
);

引数

pFont フォント へのポインタ
idx 文字幅を取得するグリフのグリフインデックス

返り値

ピクセル単位の文字幅を返します。

説明

idx が指すグリフの文字幅を取得します。
idx が指すグリフが存在しない場合や、 グリフが固有の文字幅情報を持たない場合はフォントのデフォルト文字幅情報がもつ文字幅を返します。
