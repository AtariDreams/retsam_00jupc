NNSG2dTextRect

定義


#include <nnsys/g2d/g2d_Font.h>

typedef struct NNSG2dTextRect
{
	int width;
	int height;
}
NNSG2dTextRect;

説明

NNS_G2dFontGetTextRect() および NNS_G2dTextCanvasGetTextRect() で 文字列の矩形サイズを受け取るための構造体です。

width 矩形の幅
height 矩形の高さ

参照

NNS_G2dFontGetTextRect, NNS_G2dTextCanvasGetTextRect

履歴
