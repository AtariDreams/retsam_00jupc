NNS_G2dSetAnimCtrlCallBackFunctorAtAnimFrame

構文

#include <nnsys/g2d/g2d_Animation.h>
void NNS_G2dSetAnimCtrlCallBackFunctorAtAnimFrame( 
                      NNSG2dAnimController*   pAnimCtrl, 
                      u32                     param, 
                      NNSG2dAnmCallBackPtr    pFunc 
                      u16                     frameIdx );

引数

pAnimCtrl [OUT] NNSG2dAnimController 実体
param [IN] コールバック関数の引数となる u32 データ
pFunc [IN] コールバック関数ポインタ
frameIdx [IN] コールバックを呼び出すAnimFrameのインデックス

返り値

なし。

説明

アニメーションコントローラにコールバックを設定します。
コールバックはコントローラ実体毎に1種類のみ登録可能です。

本関数の引数 type に NNS_G2D_ANMCALLBACKTYPE_SPEC_FRM 以外を指定するのは不正です。
その場合は NNS_G2dSetAnimCtrlCallBackFunctor() を 使用する必要があります。

再生フレームが変更になったタイミングで新しい再生フレームのインデックスをframeIdxと比較し
同一であればコールバックが呼び出されます。

NNS_G2dSetCallBackFunctorAtAnimFrame()から改名されました。以前の関数名は別名として残されています。
