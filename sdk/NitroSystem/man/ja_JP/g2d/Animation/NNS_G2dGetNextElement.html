NNS_G2dGetNextElement

構文

#include <nnsys/g2d/g2d_Animation.h>
void* NNS_G2dGetNextElement( 
                      NNSG2dAnimController*     pAnimCtrl );

引数

pAnimCtrl [IN] NNSG2dAnimController 実体

返り値

現在の再生フレームの次フレームのアニメーション結果先頭を指す void ポインタ

説明

現在の再生フレームの次フレームのアニメーション結果先頭を指す void ポインタを取得します。
アニメーション補間に利用します。

本関数は NNS_G2dGetAnimCtrlNextElement() に 改名されました。本関数は、過去のバージョンとの互換性維持のため、別名としてのこされます。
