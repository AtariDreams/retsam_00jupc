NNS_G2dInitAnimCtrlCallBackFunctor

構文

#include <nnsys/g2d/g2d_Animation.h>
void NNS_G2dInitAnimCtrlCallBackFunctor( 
        NNSG2dAnimController*     pAnimCtrl );

引数

pAnimCtrl [OUT] NNSG2dAnimController 実体

返り値

なし。

説明

コントローラのコールバックを初期化します。
NNS_G2dInitAnimCallBackFunctor( ) を内部で呼び出します
本関数はNNS_G2dInitAnimControllerCallBackFunctor()から改名された新しい関数名です。互換性維持のため、過去の関数は別名として残されています。

参照
