NNS_G2dStartAnimCtrl

構文

#include <nnsys/g2d/g2d_Animation.h>

  static void NNS_G2dStartAnimCtrl
                            ( 
                                NNSG2dAnimController*       pAnimCtrl 
                            );

引数

pAnimCtrl [OUT] NNSG2dAnimController 実体

返り値

なし

説明

アニメーションの再生を開始します。

注意：
STOP方式のアニメーションシーケンスを終端まで再生した場合、アニメーションコントローラが停止状態に設定されます。
そのため、再度、アニメーション更新を行うには、アニメーションコントローラに対して、NNS_G2dStartAnimCtrl()を呼び出す必要があります。

本関数はNNS_G2dStartAnimController()から改名された新しい関数名です。互換性維持のため、過去の関数は別名として残されています。
